# 資料庫讀取

資料庫的讀取與儲存，是透過 `檔案` 選單進行。

## 標準資料庫

- `讀取 AMCSD 資料庫（含 ### 個晶體）` — 讀取美國礦物學會 (MSA) 所提供的 AMCSD 資料庫。收錄約 2 萬筆晶體結構，並已隨安裝檔一併附上，因此無需另行下載。選單上會顯示目前的收錄筆數。若在 `選項` 中啟用「下次啟動時讀取 AMCSD 資料庫」，則會在啟動時自動讀取。
- `讀取 COD 資料庫（含 ### 個晶體）` — 讀取 Crystallography Open Database (COD) 所提供的晶體資料庫。收錄超過 50 萬筆晶體結構，由於檔案容量龐大（超過 800 MB）並未隨附。首次讀取時會自動從網際網路下載，之後則讀取已下載的檔案。

## 自訂資料庫

- `讀取資料庫` — 開啟任意的 CSManager 資料庫檔案 (`*.cdb3`)。
- `儲存資料庫` — 將目前的資料庫以 `*.cdb3` 格式儲存。
- `清除資料庫` — 清除已讀入的資料。資料庫檔案本身不會被刪除。

若直接使用 AMCSD 或 COD，幾乎不會用到 `讀取資料庫` / `儲存資料庫`。

## 資料庫的使用與引用

若使用 AMCSD 所收錄的晶體結構資料出版學術論文等，無論線上或離線，務必引用下列論文。

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
