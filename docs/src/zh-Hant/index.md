# CSManager 手冊

CSManager 是一款用於瀏覽、搜尋與編輯晶體結構資料庫的 Windows 桌面應用程式。它支援讀取隨附的 AMCSD 資料庫、按需取得 COD、匯入晶體檔案、編輯晶體資訊，以及與相關軟體之間的資料連動。

本 GitHub Pages 版手冊將作為今後的正式版本進行管理。初期頁面以既有 PDF/手冊的結構與現行 UI 為基礎建立了骨架，詳細內文將直接在 Markdown 端逐步補充。
