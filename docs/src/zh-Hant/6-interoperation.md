# 與其他軟體的連動

CSManager 可與 PDIndexer、ReciPro 等相關軟體交換晶體資料。

## 選取時的自動傳送

在 [晶體清單](2-database-table.md) 中選取晶體時，若 PDIndexer 或 ReciPro 已啟動，該晶體的資訊會經由剪貼簿自動傳送。只要在 CSManager 中選擇晶體，即可在目標軟體中讀入相同的晶體。

在 [晶體資訊](4-crystal-information/index.md) 上按右鍵並選擇 `Send this crystal to other software`，同樣可傳送目前的晶體。

## 晶體清單的匯入

可從 `檔案` 選單的 `匯入 PDI 晶體資料 (*.xml)`，讀入 PDIndexer 或 ReciPro 的晶體清單 (`*.xml`)（請參閱 [匯入與匯出](5-import-export.md)）。
