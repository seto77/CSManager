# 搜尋

在搜尋面板中，可依各種條件篩選已讀入的晶體資料庫。啟用要使用條件的核取方塊並輸入值，按下 `搜尋` 按鈕或按 Enter 鍵即會執行搜尋。可同時指定多個條件。

## 搜尋條件

- `Name` — 以晶體的名稱搜尋。
- `元素` — 按下 `週期表` 按鈕會開啟另一個視窗，可選擇搜尋對象的元素。各元素的按鈕每按一次便會切換狀態（可含可不含 / 必含 / 必排除）。可利用視窗上方的 "may or not include" / "must include" / "must exclude" 按鈕一次切換所有元素的狀態。
- `參考文獻` — 以論文名、期刊名、作者名等文獻資訊搜尋。
- `晶系` — 依晶系篩選。
- `晶胞參數` — 指定晶胞參數與容許範圍。容許範圍輸入為零的項目會被忽略。
- `晶面間距` — 指定晶面的 d 值（晶面間距）與容許範圍，搜尋具有落在該範圍內 d 值的晶體。若啟用 `忽略散射因子`，則不考慮繞射強度（晶體結構因子），連違反消光法則的晶面也一併納入搜尋。若停用，則考慮晶體結構因子，僅以強度前 20 名的晶面為搜尋對象。
- `密度` — 指定密度與容許範圍。
