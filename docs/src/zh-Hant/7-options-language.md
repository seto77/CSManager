# 選項與語言

## 選項

`選項` 選單中有下列設定。

- `工具提示` — 啟用後，會在各控制項上顯示工具提示（說明的提示框）。
- `下次啟動時讀取 AMCSD 資料庫` — 啟用後，會在啟動的同時讀取 AMCSD 資料庫。預設為啟用。
- `漸進式搜尋` — 啟用後，在變更搜尋條件的當下便立即反映到搜尋結果。
- `資料表字型大小` — 變更晶體清單的字型大小。

## 說明

`說明` 選單中有下列項目。

- `程式更新` — 確認是否已發布新版本，若有則進行更新。
- `版本紀錄` — 顯示至今的版本紀錄。
- `授權` — 顯示授權。
- `Github 頁面` — 開啟 GitHub 上的 CSManager 儲存庫。
- `回報錯誤、需求或意見` — 開啟 GitHub 的 Issue 頁面。
- `說明 (PDF)` — 以預設瀏覽器開啟這份線上手冊。

## 語言

可從 `語言` 選單切換顯示語言。切換時需要重新啟動（重新啟動會自動進行）。CSManager 除英文、日文外，還支援德文、法文、西班牙文、葡萄牙文、義大利文、俄文、中文（簡體字與繁體字）、韓文。
