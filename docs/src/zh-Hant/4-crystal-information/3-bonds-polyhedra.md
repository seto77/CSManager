# 鍵結與配位多面體

在 `鍵 & 多面體` 分頁中，可設定以三維顯示晶體結構時的鍵結與配位多面體繪製。在此定義的鍵結與多面體，會用於 ReciPro 等結構檢視器中的顯示。

## 鍵結的定義

- `成鍵原子` — 選擇要成鍵的 2 種原子（中心原子與頂點原子）。
- `新增` / `取代` / `刪除` — 將設定的鍵結新增至清單、更新所選的鍵結或將其刪除。

## 顯示選項

- `顯示鍵` — 顯示原子間的鍵結。
- `顯示多面體` — 顯示配位多面體。
- `顯示稜邊` — 顯示多面體的稜邊。
- `顯示頂點原子` / `顯示中心原子` — 顯示多面體的頂點原子與中心原子。
- `顯示內部鍵` — 顯示多面體內部的鍵結。

多面體的透明度等細部繪製設定，也可在此分頁中調整。
