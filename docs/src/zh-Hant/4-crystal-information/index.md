# 晶體資訊

在晶體資訊區域（`Crystal Information` 控制項）中，可顯示與編輯所選晶體的晶胞參數、對稱性、原子位置、文獻等。也可以將 CIF 檔案或 AMC 檔案拖放到此區域，以讀入任意晶體。

!!! warning "變更請以 `新增` / `取代` 確定"
    對晶體進行變更後，務必按下 `新增` 或 `取代` 按鈕。若未按下，變更內容不會儲存至晶體清單而會遺失。

最上方會顯示晶體名 (`Name`) 與化學組成 (`Formula`) 等共通資訊，可用 `Reset` 將晶體的所有資訊初始化。下方排列著可設定對稱性、原子、文獻等的分頁。

## 分頁

- [基本資訊](1-basic-information.md) — 晶胞參數、對稱性、體積、密度等（`基本資訊` 分頁）
- [原子](2-atoms.md) — 原子位置、佔有率、溫度因子、散射因子（`原子資訊` 分頁）
- [鍵結與配位多面體](3-bonds-polyhedra.md) — 鍵結與配位多面體的繪製設定（`鍵 & 多面體` 分頁）
- [文獻](4-references.md) — 引用文獻資訊（`文獻` 分頁）

## 右鍵選單

在控制項的空白處按右鍵，可進行下列操作。

- `從 CIF 或 AMC 匯入...` — 以 CIF / AMC 格式匯入晶體結構。
- `匯出為 CIF` — 將目前的晶體以 CIF 格式輸出。
- `Send this crystal to other software` — 將所選晶體傳送至 PDIndexer 或 ReciPro 等。
- `對稱性資訊` — 開啟與對稱性相關的詳細視窗（空間群的流水編號、勞厄群與點群、出現法則、Wyckoff position、晶面與晶帶軸的幾何計算等）。
- `還原晶胞常數` — 將晶胞參數還原為剛讀入時的值。
- `轉換為 P1 空間群` / `轉換為超結構` / `變更軸/原點設定` — 進行空間群或晶格的轉換。
