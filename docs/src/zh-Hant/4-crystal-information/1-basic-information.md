# 基本資訊

在 `基本資訊` 分頁中，可顯示與設定晶體的晶胞參數與對稱性。

## 晶胞參數與對稱性

- **Cell constants** — 顯示與設定晶胞參數。長度的單位為 Å (10⁻¹⁰ m)。當指定對稱性而使晶胞參數受到限制時（例如：立方晶的 a = b = c, α = β = γ = 90°），會自動重新設定。
- **Symmetry** — 依各階層設定對稱性。指定 `Crystal system`（晶系）、`Point group`（點群）、`Space group`（空間群）。空間群請留意軸的設定。在 `Search` 中輸入空間群字串後，候選會顯示於右側清單中（區分大小寫）。

## 計算得出的值

若已輸入原子資訊，則會計算並顯示下列值。

- **Cell Volume** — 單位晶胞的體積。
- **Cell Mass** — 單位晶胞的質量。
- **Molar Volume** / **Molar Mass** — 每莫耳的體積與質量。
- **Z** — 單位晶胞中所含組成式的數目。
- **Density** — 密度。
- **Color of Profile** — 繪製繞射峰的顏色。點選後會開啟顏色設定視窗。

更詳細的對稱性資訊（勞厄群與點群的各種表示法、出現法則、Wyckoff position、晶面與晶帶軸的幾何計算），可從右鍵選單的 `對稱性資訊` 確認。
