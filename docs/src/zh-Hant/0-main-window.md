# 主視窗

啟動 CSManager 後會顯示主視窗。視窗大致由下列區域構成。

- **選單（最上方）** — `檔案`、`選項`、`說明`、`語言` 各選單。可進行資料庫的讀寫與程式設定。
- **晶體清單（中央）** — 已讀入晶體的清單。選取列後會反映到下方的晶體資訊。
- **晶體資訊（下方）** — 顯示與編輯所選晶體的晶胞參數、對稱性、原子位置等。
- **搜尋面板（下方右側）** — 依各種條件篩選已讀入的資料庫。
- **狀態區（最下方）** — 顯示讀取進度與處理結果的訊息。
