# Options and language

## Option

The `Option` menu has the following settings.

- `Tool Tip` — when enabled, shows tooltips (explanatory balloons) on each control.
- `Read AMCSD Database on Next Boot` — when enabled, reads the AMCSD database at startup. It is enabled by default.
- `Incremental Search` — when enabled, updates the search results immediately whenever a search condition is changed.
- `Font Size in Data Table` — changes the font size of the database table.

## Help

The `Help` menu has the following items.

- `Program Updates` — checks whether a new version has been released and, if so, updates to it.
- `Version History` — shows the past version history.
- `License` — shows the license.
- `Github page` — opens the CSManager repository on GitHub.
- `Repport bugs, request or comments` — opens the GitHub Issue page.
- `Help (PDF)` — opens this online manual in your default browser.

## Language

You can switch the display language from the `Language` menu. Switching requires a restart (the restart is performed automatically). In addition to English and Japanese, CSManager supports German, French, Spanish, Portuguese, Italian, Russian, Chinese (Simplified and Traditional), and Korean.
