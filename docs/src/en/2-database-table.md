# Database table

When you load a database, a list of crystals (the database table) appears in the center of the main window. Each row corresponds to a single crystal, and information such as the name, chemical composition, symmetry, cell constants, density, and reference is shown in the columns.

## Selection

When you select a crystal, its details are shown in the [Crystal information](4-crystal-information/index.md) area at the bottom. At the same time, if PDIndexer or ReciPro is running, the crystal's information is transferred to that software via the clipboard the moment it is selected (see [Interoperation](6-interoperation.md)).

## Sorting

Clicking a column header (at the top) sorts the table in ascending or descending order by that item.
