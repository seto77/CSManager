# Database Loading

CSManager can load the bundled AMCSD database and the optional COD database.

## Built-in Databases

- `Read AMSCD Database (including ### crystals)` loads the bundled AMCSD dataset.
- `Read COD Database (including ### crystals)` loads COD data. The first use may require downloading the large COD archive.

## Custom Databases

- `Read Database` opens an existing CSManager database file.
- `Save Database` saves the current database.
- `Clear Database` clears the loaded entries without deleting the database files.
