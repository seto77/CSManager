# CSManager Manual

CSManager is a Windows desktop application for searching, browsing, editing, and exporting the crystal structures provided by the American Mineralogist Crystal Structure Database (AMCSD) and the Crystallography Open Database (COD). It ships with the AMCSD database of about 20,000 entries, and the COD database of more than 500,000 entries can also be obtained on demand. Loaded crystals can be imported and exported in CIF / AMC format, and a selected crystal can be sent to related software such as PDIndexer and ReciPro for interoperation.
