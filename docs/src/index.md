# CSManager Manual

CSManager is a Windows desktop application for browsing, searching, and maintaining crystal structure databases. It can load the bundled AMCSD database, download and load COD on demand, import common crystal formats, edit crystal information, and exchange selected crystal data with related tools.

This GitHub Pages manual is the new canonical documentation source. The initial pages are scaffolded from the existing PDF/manual structure and the current application UI; detailed content will be expanded directly in Markdown.
