# Руководство CSManager

CSManager — это настольное приложение для Windows, предназначенное для поиска, просмотра, редактирования и экспорта кристаллических структур, предоставляемых American Mineralogist Crystal Structure Database (AMCSD) и Crystallography Open Database (COD). В комплект входит база AMCSD, содержащая около 20 000 записей, а базу COD, насчитывающую более 500 000 записей, можно загрузить по запросу. Загруженные кристаллы можно импортировать и экспортировать в форматах CIF / AMC, а выбранный кристалл — передавать в сопутствующие программы, такие как PDIndexer и ReciPro, для совместной работы.

Это бесплатное программное обеспечение, распространяемое на условиях [лицензии MIT](https://github.com/seto77/CSManager/blob/master/LICENSE.md). Программа предоставляется без каких-либо гарантий, и автор не несёт ответственности за любые проблемы, возникшие в результате её использования.

Для работы требуется ОС Windows, на которой может работать **.NET Desktop Runtime 8** или выше.
