# Руководство CSManager

CSManager — это настольное приложение Windows для просмотра, поиска и редактирования баз данных кристаллических структур. Оно может загружать встроенную базу данных AMCSD, по требованию скачивать и загружать COD, импортировать распространённые форматы кристаллов, редактировать сведения о кристаллах и обмениваться выбранными данными о кристаллах с сопутствующими программами.

Эта версия руководства на GitHub Pages является новым основным источником документации. Начальные страницы построены на основе структуры существующего PDF/руководства и текущего интерфейса приложения; подробное содержание будет дополняться непосредственно в Markdown.
