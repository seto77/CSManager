# Взаимодействие с другими программами

CSManager может обмениваться данными о кристаллах с сопутствующими инструментами, такими как PDIndexer и ReciPro.

## Планируемое содержание

- Передача через буфер обмена при выборе кристалла.
- Импорт списков кристаллов из PDIndexer или ReciPro.
- Ожидаемые форматы файлов и данных.
