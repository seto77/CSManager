# Список кристаллов

В списке кристаллов перечислены загруженные записи кристаллов. Выбор строки обновляет редактор сведений о кристалле.

## Планируемое содержание

- Сортировка столбцов.
- Фильтрация записей AMCSD и COD.
- Чтение столбцов химической формулы, плотности, параметров решётки, симметрии и литературных ссылок.
- Отправка выбранного кристалла в сопутствующие приложения.
