# Список кристаллов

После загрузки базы данных в центральной части главного окна отображается перечень кристаллов (список кристаллов). Каждая строка соответствует одному кристаллу, а в столбцах отображаются такие сведения, как название, химический состав, симметрия, параметры ячейки, плотность и литература.

## Выбор

При выборе кристалла его подробные данные отображаются в области [Сведения о кристалле](4-crystal-information/index.md) внизу. Кроме того, если запущены PDIndexer или ReciPro, при выборе данные кристалла одновременно передаются в эти программы через буфер обмена (см. [Взаимодействие с другими программами](6-interoperation.md)).

## Сортировка

При нажатии на заголовок столбца (в самом верху) записи сортируются по этому полю по возрастанию или убыванию.
