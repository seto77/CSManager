# Параметры и язык

На этой странице рассматриваются параметры приложения, настройки отображения и переключение языка.

## Планируемое содержание

- Подсказки.
- Загрузка базы данных AMCSD при запуске.
- Инкрементальный поиск.
- Размер шрифта в таблице данных.
- Переключение языка и поведение при перезапуске.
- Пункты меню справки.
