# Параметры и язык

## Параметры

В меню `Параметры` доступны следующие настройки.

- `Подсказки` — при включении для каждого элемента управления отображаются подсказки (всплывающие пояснения).
- `Загружать базу AMCSD при следующем запуске` — при включении база AMCSD загружается сразу при запуске. По умолчанию включено.
- `Инкрементальный поиск` — при включении результаты поиска обновляются сразу же при изменении условий поиска.
- `Размер шрифта в таблице данных` — изменяет размер шрифта в списке кристаллов.

## Справка

В меню `Справка` доступны следующие пункты.

- `Обновления программы` — проверяет, выпущена ли новая версия, и при наличии обновляет программу.
- `История версий` — отображает историю предыдущих версий.
- `Лицензия` — отображает лицензию.
- `Страница Github` — открывает репозиторий CSManager на GitHub.
- `Сообщить об ошибках, пожеланиях или комментариях` — открывает страницу Issue на GitHub.
- `Справка (PDF)` — открывает это онлайн-руководство в браузере по умолчанию.

## Язык

В меню `Язык` можно переключить язык интерфейса. Для переключения требуется перезапуск (он выполняется автоматически). Помимо английского и японского, CSManager поддерживает немецкий, французский, испанский, португальский, итальянский, русский, китайский (упрощённый и традиционный) и корейский языки.
