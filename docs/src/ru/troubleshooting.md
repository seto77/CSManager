# Устранение неполадок

На этой странице собраны распространённые проблемы и способы их устранения.

## Планируемое содержание

- Сбои при скачивании или загрузке COD.
- Отсутствие .NET Desktop Runtime.
- Файлы базы данных, которые не удаётся открыть.
- Медленный поиск или поведение при работе с большой базой данных.
- Проблемы при обмене данными с PDIndexer или ReciPro.
