# Загрузка базы данных

Загрузка и сохранение базы данных выполняются через меню `Файл`.

## Стандартные базы данных

- `Загрузить базу AMCSD (кристаллов: ###)` — загружает базу AMCSD, предоставляемую Минералогическим обществом Америки (MSA). Она содержит около 20 000 кристаллических структур и входит в комплект установки, поэтому отдельная загрузка не требуется. В меню отображается текущее количество записей. Если в `Параметры` включён пункт «Загружать базу AMCSD при следующем запуске», база загружается автоматически при запуске.
- `Загрузить базу COD (кристаллов: ###)` — загружает базу кристаллов, предоставляемую Crystallography Open Database (COD). Она содержит более 500 000 кристаллических структур и из-за большого размера файла (более 800 МБ) не входит в комплект. При первой загрузке файл автоматически скачивается из интернета, а при последующих загрузках используется уже скачанный файл.

## Пользовательские базы данных

- `Загрузить базу данных` — открывает произвольный файл базы данных CSManager (`*.cdb3`).
- `Сохранить базу данных` — сохраняет текущую базу данных в формате `*.cdb3`.
- `Очистить базу данных` — очищает загруженные данные. Сам файл базы данных при этом не удаляется.

Если вы используете AMCSD или COD в неизменном виде, пункты `Загрузить базу данных` / `Сохранить базу данных` практически не понадобятся.

## Использование базы данных и цитирование

Если вы используете данные кристаллических структур из AMCSD для публикации научных статей и т. п., независимо от способа (онлайн или офлайн), обязательно цитируйте следующую работу.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
