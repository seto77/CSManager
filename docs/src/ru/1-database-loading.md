# Загрузка базы данных

CSManager может загружать встроенную базу данных AMCSD и дополнительную базу данных COD.

## Встроенные базы данных

- `Загрузить базу AMCSD (кристаллов: ###)` загружает встроенный набор данных AMCSD.
- `Загрузить базу COD (кристаллов: ###)` загружает данные COD. При первом использовании может потребоваться скачивание большого архива COD.

## Пользовательские базы данных

- `Загрузить базу данных` открывает существующий файл базы данных CSManager.
- `Сохранить базу данных` сохраняет текущую базу данных.
- `Очистить базу данных` очищает загруженные записи, не удаляя сами файлы базы данных.
