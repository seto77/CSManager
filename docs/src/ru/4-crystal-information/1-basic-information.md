# Основные сведения

На вкладке `Осн. инфо` отображаются и задаются параметры ячейки и симметрия кристалла.

## Параметры ячейки и симметрия

- **Cell constants** — отображение и задание параметров ячейки. Единица длины — Å (10⁻¹⁰ m). Если из-за заданной симметрии на параметры ячейки накладываются ограничения (например, для кубической системы a = b = c, α = β = γ = 90°), они задаются автоматически.
- **Symmetry** — задание симметрии по уровням. Указываются `Crystal system` (кристаллическая система), `Point group` (точечная группа) и `Space group` (пространственная группа). Обращайте внимание на установку осей пространственной группы. Если ввести строку с обозначением пространственной группы в поле `Search`, в списке справа отобразятся подходящие варианты (с учётом регистра).

## Вычисляемые значения

Если введены данные об атомах, вычисляются и отображаются следующие значения.

- **Cell Volume** — объём элементарной ячейки.
- **Cell Mass** — масса элементарной ячейки.
- **Molar Volume** / **Molar Mass** — объём и масса на один моль.
- **Z** — число формульных единиц в элементарной ячейке.
- **Density** — плотность.
- **Color of Profile** — цвет для отрисовки дифракционных пиков. При нажатии открывается окно настройки цвета.

Более подробные сведения о симметрии (различные обозначения групп Лауэ и точечных групп, правила появления отражений, Wyckoff position, геометрические расчёты для кристаллических плоскостей и осей зон) можно посмотреть через пункт `Информация о симметрии` контекстного меню.
