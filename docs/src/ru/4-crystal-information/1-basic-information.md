# Основные сведения

На этой странице рассматриваются основные поля кристалла, включая название, химическую формулу, параметры элементарной ячейки, объём, плотность, кристаллическую систему, точечную группу и пространственную группу.

## Планируемое содержание

- Редактирование параметров решётки и углов.
- Обновление значений, связанных с химической формулой.
- Проверка сведений о симметрии.
- Применение изменений с помощью кнопки `Добавить` или `Заменить`.
