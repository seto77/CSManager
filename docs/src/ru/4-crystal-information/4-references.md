# Литература

На вкладке `Лит.` отображаются и задаются библиографические данные источника, из которого взяты данные кристаллической структуры.

- `Note` — заметки и примечания.
- `Authors` — фамилии авторов цитируемой статьи.
- `Journal` — название журнала цитируемой статьи.
- `Title` — заголовок цитируемой статьи.
