# Главное окно

При запуске CSManager отображается главное окно. В целом оно состоит из следующих областей.

- **Меню (вверху)** — меню `Файл`, `Параметры`, `Справка` и `Язык`. Служат для чтения и записи базы данных, а также для настройки приложения.
- **Список кристаллов (в центре)** — перечень загруженных кристаллов. При выборе строки её данные отображаются в области сведений о кристалле внизу.
- **Сведения о кристалле (внизу)** — отображение и редактирование параметров ячейки, симметрии, положений атомов и других данных выбранного кристалла.
- **Панель поиска (внизу справа)** — фильтрация загруженной базы данных по различным условиям.
- **Область состояния (в самом низу)** — отображение хода загрузки и сообщений о результатах операций.
