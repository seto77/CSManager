# Главное окно

Главное окно построено вокруг списка кристаллов, сведений о выбранном кристалле и панели поиска.

## Области

- Меню файлов и меню приложения в верхней части.
- Список кристаллов в центральной области.
- Редактор сведений о кристалле для просмотра и редактирования выбранной записи.
- Панель поиска для фильтрации загруженной базы данных.
- Область состояния для отображения хода выполнения и сообщений.

## Заметки по источнику

- Подписи интерфейса следует сверять с файлом `CSManager/FormMain.ru.resx`.
- Снимки экрана переносятся в `docs/src/assets/cap-ru-auto/` или `docs/src/assets/cap-ru-manual/`.
