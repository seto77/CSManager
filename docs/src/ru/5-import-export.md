# Импорт и экспорт

CSManager может импортировать данные о кристаллах из файлов CIF, AMC и XML PDIndexer/ReciPro. Он может экспортировать выбранный кристалл в формате CIF и сохранять подготовленные базы данных в виде файлов базы данных CSManager.

## Планируемое содержание

- `Импорт CIF, AMC …`
- `Импорт данных кристаллов PDI (*.xml)`
- Импорт перетаскиванием в область сведений о кристалле.
- Сохранение баз данных.
- Экспорт выбранных кристаллов.
