# Поиск

На панели поиска можно фильтровать загруженную базу кристаллов по различным условиям. Установите флажок нужного условия, введите значение и нажмите кнопку `Поиск` или клавишу Enter, чтобы выполнить поиск. Можно задавать несколько условий одновременно.

## Условия поиска

- `Name` — поиск по названию кристалла.
- `Элементы` — при нажатии кнопки `Таблица Менделеева` открывается отдельное окно, в котором можно выбрать искомые элементы. При каждом нажатии кнопки элемента её состояние переключается (может содержаться / должен содержаться / должен быть исключён). Кнопками "may or not include" / "must include" / "must exclude" в верхней части окна можно одновременно переключить состояние всех элементов.
- `Ссылка` — поиск по библиографическим данным: названию статьи, названию журнала, фамилиям авторов и т. п.
- `Крист. система` — фильтрация по кристаллической системе.
- `Парам. ячейки` — задание параметров ячейки и допуска. Поля, в которых допуск равен нулю, игнорируются.
- `d-расст.` — задание d-расстояния (межплоскостного расстояния) кристаллической плоскости и допуска; ищутся кристаллы, имеющие значения d в этом диапазоне. Если включён параметр `Игнорировать фактор рассеяния`, интенсивность дифракции (структурный фактор кристалла) не учитывается, и в поиск включаются также плоскости, нарушающие правила погасания. Если он выключен, структурный фактор учитывается, и поиск ведётся только по 20 плоскостям с наибольшей интенсивностью.
- `Плотность` — задание плотности и допуска.
