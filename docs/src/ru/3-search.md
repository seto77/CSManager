# Поиск

Панель поиска фильтрует загруженную базу данных кристаллов по названию, составу, симметрии, параметрам решётки, межплоскостным расстояниям (d), плотности и литературным сведениям.

## Планируемое содержание

- Текстовый поиск по названию и литературным ссылкам.
- Включение и исключение элементов с помощью диалога периодической таблицы.
- Фильтрация по кристаллической системе и пространственной группе.
- Диапазоны параметров решётки.
- Поиск по межплоскостному расстоянию (d) с учётом и без учёта фактора рассеяния.
- Поиск по диапазону плотности.
