# 結晶リスト

データベースを読み込むと、メインウィンドウの中央部に結晶の一覧（結晶リスト）が表示されます。各行が 1 件の結晶に対応し、名称・化学組成・対称性・格子定数・密度・文献などの情報が列に表示されます。

## 選択

結晶を選択すると、その結晶の詳細が下部の [結晶情報](4-crystal-information/index.md) に表示されます。あわせて、PDIndexer や ReciPro が起動している場合は、選択と同時にその結晶の情報がクリップボード経由でこれらのソフトへ転送されます（[他ソフトとの連携](6-interoperation.md) を参照）。

## 並び替え

各列の見出し（最上部）をクリックすると、その項目で昇順・降順に並び替えられます。
