# データベース読込

CSManager は、バンドルされた AMCSD データベースと、必要に応じて取得する COD データベースを読み込めます。

## 標準データベース

- `AMCSD データベースを読込 (データ数: ###)` は、同梱された AMCSD データセットを読み込みます。
- `CODデータベースを読込 (データ数: ###)` は、COD データを読み込みます。初回は大きな COD アーカイブのダウンロードが必要になる場合があります。

## カスタムデータベース

- `データベースを読込` は既存の CSManager データベースファイルを開きます。
- `データベースを保存` は現在のデータベースを保存します。
- `データベースをクリア` は読み込み済みの項目をクリアします。データベースファイル自体は削除されません。
