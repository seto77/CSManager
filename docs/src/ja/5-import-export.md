# インポートとエクスポート

CSManager は、CIF・AMC・PDIndexer / ReciPro の結晶リストから結晶をインポートでき、選択中の結晶を CIF として書き出せます。

## インポート

- `ファイル` メニューの `インポート (単一ファイル) (CIF, AMC)` — CIF ファイルや AMC ファイルを読み込み、現在のデータベースの末尾に追加します。
- `ファイル` メニューの `インポート (PDIndexer, ReciProの結晶リスト) (*.xml)` — PDIndexer や ReciPro の結晶リスト (`*.xml`) を読み込み、現在のデータベースの末尾に追加します。
- [結晶情報](4-crystal-information/index.md) 領域に CIF / AMC ファイルをドラッグ＆ドロップしても、結晶を読み込めます。また、結晶情報を右クリックして `CIFファイル又はAMCファイルから読み込み` を選んでもインポートできます。

CIF は国際結晶学連合 (IUCr) の標準フォーマット、AMC はアメリカ鉱物学会が採用するフォーマットです。

## エクスポート

- [結晶情報](4-crystal-information/index.md) を右クリックして `CIFファイルとして出力` を選ぶと、現在の結晶を CIF 形式で書き出せます。
- データベース全体を保存するには、`ファイル` メニューの `データベースを保存` を使います（[データベース読込](1-database-loading.md) を参照）。
