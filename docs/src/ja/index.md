# CSManager マニュアル

CSManager は、American Mineralogist Crystal Structure Database (AMCSD) や Crystallography Open Database (COD) が提供する結晶構造を、検索・閲覧・編集・出力するための Windows デスクトップアプリケーションです。約 2 万件を収録した AMCSD データベースを同梱しており、50 万件を超える COD データベースもオンデマンドで取得できます。読み込んだ結晶は CIF / AMC 形式でインポート・エクスポートでき、選択した結晶を PDIndexer や ReciPro などの関連ソフトへ転送して連携できます。
