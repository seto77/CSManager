# CSManager マニュアル

CSManager は、結晶構造データベースを閲覧・検索・編集するための Windows デスクトップアプリケーションです。バンドルされた AMCSD データベースの読み込み、COD のオンデマンド取得、結晶ファイルのインポート、結晶情報の編集、関連ソフトとのデータ連携に対応します。

この GitHub Pages 版マニュアルを、今後の正本として管理します。初期ページは既存 PDF/manual の構成と現行 UI をもとに骨組みを用意しており、詳細本文は Markdown 側へ直接追記していきます。
