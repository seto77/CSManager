# 結晶情報

結晶情報領域（`Crystal Information` コントロール）では、選択中の結晶の格子定数・対称性・原子位置・文献などを表示・編集します。この領域に CIF ファイルや AMC ファイルをドラッグ＆ドロップして、任意の結晶を読み込むこともできます。

!!! warning "変更は `追加` / `入れ替え` で確定"
    結晶に変更を加えた場合は、必ず `追加` または `入れ替え` ボタンを押してください。押さなかった場合、変更内容は結晶リストに保存されず失われます。

最上部には結晶名 (`Name`) と化学組成 (`Formula`) などの共通情報が表示され、`Reset` で結晶の全情報を初期化できます。下部には、対称性・原子・文献などを設定するタブが並んでいます。

## タブ

- [基本情報](1-basic-information.md) — 格子定数・対称性・体積・密度など（`格子/対称性` タブ）
- [原子](2-atoms.md) — 原子位置・占有率・温度因子・散乱因子（`原子情報` タブ）
- [結合と配位多面体](3-bonds-polyhedra.md) — 結合と配位多面体の描画設定（`多面体/結合描画設定` タブ）
- [文献](4-references.md) — 引用文献情報（`引用文献` タブ）

## 右クリックメニュー

コントロールの空白部分を右クリックすると、次の操作が行えます。

- `CIFファイル又はAMCファイルから読み込み` — 結晶構造を CIF / AMC 形式でインポートします。
- `CIFファイルとして出力` — 現在の結晶を CIF 形式で書き出します。
- `Send this crystal to other software` — 選択中の結晶を PDIndexer や ReciPro などへ送ります。
- `対称性詳細` — 対称性に関する詳細ウィンドウ（空間群の通し番号、ラウエ群・点群、出現則、Wyckoff position、結晶面・晶帯軸の幾何計算など）を開きます。
- `アプリ起動時の格子定数に戻す` — 格子定数を読み込み直後の値に戻します。
- `P1に変換` / `超構造に変換` / `異なる軸・原点セッティングの空間群に変換` — 空間群や格子の変換を行います。
