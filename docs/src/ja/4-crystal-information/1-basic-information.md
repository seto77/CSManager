# 基本情報

`格子/対称性` タブでは、結晶の格子定数と対称性を表示・設定します。

## 格子定数と対称性

- **Cell constants** — 格子定数を表示・設定します。長さの単位は Å (10⁻¹⁰ m) です。対称性を指定して格子定数に制約がある場合（例: 立方晶の a = b = c, α = β = γ = 90°）は、自動的に再設定されます。
- **Symmetry** — 対称性を階層ごとに設定します。`Crystal system`（結晶系）、`Point group`（点群）、`Space group`（空間群）を指定します。空間群は軸のセッティングに注意してください。`Search` に空間群の文字列を入力すると、候補が右のリストに表示されます（大文字・小文字を区別します）。

## 算出される値

原子情報が入力されている場合、次の値が計算されて表示されます。

- **Cell Volume** — 単位格子の体積。
- **Cell Mass** — 単位格子の質量。
- **Molar Volume** / **Molar Mass** — 1 モルあたりの体積・質量。
- **Z** — 単位格子に含まれる組成式の数。
- **Density** — 密度。
- **Color of Profile** — 回折ピークを描画する色。クリックすると色設定ウィンドウが開きます。

より詳しい対称性情報（ラウエ群・点群の各種表記、出現則、Wyckoff position、結晶面・晶帯軸の幾何計算）は、右クリックメニューの `対称性詳細` から確認できます。
