# 原子

`原子情報` タブでは、結晶に含まれる原子の情報を表示・設定します。上部のリストに原子の一覧が表示され、各原子をクリックすると詳しい情報が下部に表示されます。

!!! warning
    原子を編集したら、`Add` または `Replace` ボタンを押してください。押さないと原子情報がリストに保存されません。

## 原子リストの操作

- `Add` — 設定した原子をリストに新規追加します。
- `Replace` — 設定した原子を現在選択中の原子と入れ替えます。
- `Up` / `Down` — 選択した原子の順番を上下に移動します。
- `Delete` — 選択した原子をリストから削除します。

## 元素と位置

- `Label` — 原子のラベルを入力します。
- `Element` — 元素を設定します。
- `X`, `Y`, `Z` — 原子の分率座標を設定します。0 から 1 までの実数を入力します。1/2 や 2/3 のような分数も入力できます。
- `Occ` — 占有率を 0 から 1 までの実数で指定します。

**Origin shift** では、プリセットボタンや任意の値で原点位置をシフトできます。

## 温度因子（Debye-Waller factor）

- **Notation** — `U` か `B` を選択します。
- **Model** — 等方性 (Isotropy) か異方性 (Anisotropy) を選択します。
- 温度因子の各成分（`B##` または `U##`）を入力します。

## 散乱因子（Scattering factor）

原子散乱因子を計算する際の価数や同位体組成を設定します。

- **X-ray** — X 線に対する弾性原子散乱因子の計算に用いる価数を選択します。パラメータは *International Tables for Crystallography volume C* に基づきます。
- **Electron** — 電子線に対する弾性原子散乱因子の価数を選択します。パラメータは Peng (1998, *Acta Cryst.* A54, 481-485) に基づきます。
- **Neutron** — 中性子の弾性散乱長を計算する同位体組成を選択します。`Natural isotope abundance`（天然存在比）または `Custom isotope abundance`（任意設定）を選べます。
