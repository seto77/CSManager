# 結合と配位多面体

`多面体/結合描画設定` タブでは、結晶構造を 3 次元表示する際の結合と配位多面体の描画を設定します。ここで定義した結合・多面体は、ReciPro などの構造ビューアでの表示に用いられます。

## 結合の定義

- `結合原子` — 結合させる 2 種類の原子（中心原子と頂点原子）を選びます。
- `追加` / `更新` / `削除` — 設定した結合をリストに追加、選択中の結合を更新、または削除します。

## 表示オプション

- `結合を表示` — 原子間の結合を表示します。
- `多面体を表示` — 配位多面体を表示します。
- `辺を表示` — 多面体の稜（辺）を表示します。
- `頂点を表示` / `中心を表示` — 多面体の頂点原子・中心原子を表示します。
- `結合を表示`（内部結合） — 多面体内部の結合を表示します。

多面体の透明度などの細かな描画設定も、このタブで調整できます。
