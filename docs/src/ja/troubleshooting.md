# トラブルシューティング

よくある問題と対処方法をまとめます。

- **起動できない / .NET ランタイムが見つからない** — CSManager の動作には **.NET Desktop Runtime 8** 以上が必要です。インストールされていない場合は、Microsoft の配布ページから入手してください。
- **COD の読み込みに失敗する** — `COD データベースを読込` は初回にインターネットから大容量ファイル（800 MB 超）をダウンロードします。ネットワーク接続と空きディスク容量を確認し、時間をおいて再試行してください。
- **データベースファイルを開けない** — `*.cdb3` ファイルが破損・移動・別バージョンで保存されていないかを確認してください。AMCSD / COD をそのまま使う場合は、`AMCSD データベースを読込` / `COD データベースを読込` から読み直せます。
- **大きなデータベースで動作が遅い** — COD のような大規模データベースでは検索や読み込みに時間がかかることがあります。必要な条件で絞り込んでから操作してください。
- **PDIndexer / ReciPro と連携できない** — 連携先のソフトが起動しているかを確認してください。選択時の自動転送はクリップボード経由で行われます（[他ソフトとの連携](6-interoperation.md) を参照）。

ここに記載のない問題は、GitHub の [Issue](https://github.com/seto77/CSManager/issues) へご報告ください。
