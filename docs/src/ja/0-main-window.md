# メインウィンドウ

メインウィンドウは、結晶リスト、選択中の結晶情報、検索パネルを中心に構成されています。

## 領域

- 上部のファイルメニューとアプリケーションメニュー。
- 中央の結晶リスト。
- 選択した結晶を確認・編集する結晶情報エディタ。
- 読み込んだデータベースを絞り込む検索パネル。
- 進捗やメッセージを表示するステータス領域。
