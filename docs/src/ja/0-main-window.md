# メインウィンドウ

CSManager を起動すると、メインウィンドウが表示されます。ウィンドウは大きく分けて、次の領域から構成されています。

- **メニュー（最上部）** — `ファイル`・`オプション`・`ヘルプ`・`言語` の各メニュー。データベースの読み書きやアプリ設定を行います。
- **結晶リスト（中央部）** — 読み込んだ結晶の一覧。行を選択すると下部の結晶情報に反映されます。
- **結晶情報（下部）** — 選択中の結晶の格子定数・対称性・原子位置などを表示・編集します。
- **検索パネル（下部右側）** — 読み込んだデータベースを各種条件で絞り込みます。
- **ステータス領域（最下部）** — 読み込み進捗や処理結果のメッセージを表示します。
