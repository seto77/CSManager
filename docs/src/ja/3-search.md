# 検索

検索パネルでは、読み込んだ結晶データベースを各種条件で絞り込めます。使いたい条件のチェックボックスを有効にして値を入力し、`検索` ボタンを押すか Enter キーを押すと検索が実行されます。複数の条件を同時に指定できます。

## 検索条件

- `名前` — 結晶の名称で検索します。
- `元素` — `周期表` ボタンを押すと別ウィンドウが開き、検索対象の元素を選択できます。各元素のボタンは押すごとに状態（含んでもよい / 必ず含む / 必ず除外）が切り替わります。ウィンドウ上部の "may or not include" / "must include" / "must exclude" ボタンで全元素の状態を一括で切り替えられます。
- `文献` — 論文名・雑誌名・著者名などの文献情報で検索します。
- `結晶系` — 結晶系で絞り込みます。
- `格子定数` — 格子定数と許容範囲を指定します。許容範囲にゼロを入力した項目は無視されます。
- `d値` — 結晶面の d 値（面間隔）と許容範囲を指定し、その範囲に含まれる d 値を持つ結晶を検索します。`散乱因子を無視` を有効にした場合は、回折強度（結晶構造因子）を考慮せず、消滅則に抵触する結晶面も含めて検索します。無効の場合は結晶構造因子を考慮し、強度上位 20 までの結晶面だけを対象に検索します。
- `密度` — 密度と許容範囲を指定します。
