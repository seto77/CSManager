# オプションと言語

## オプション

`オプション` メニューには次の設定があります。

- `ツールチップを表示` — 有効にすると、各コントロールにツールチップ（説明の吹き出し）を表示します。
- `起動時にAMCSDデータベースを自動で読み込む` — 有効にすると、起動と同時に AMCSD データベースを読み込みます。既定で有効です。
- `Incremental Search` — 有効にすると、検索条件を変更した時点で即座に検索結果へ反映します。
- `Font Size in Data Table` — 結晶リストの文字サイズを変更します。

## ヘルプ

`ヘルプ` メニューには次の項目があります。

- `アップデート` — 新しいバージョンがリリースされているかを確認し、あればアップデートします。
- `バージョン履歴` — これまでのバージョン履歴を表示します。
- `ライセンス` — ライセンスを表示します。
- `Github ページ` — GitHub 上の CSManager リポジトリを開きます。
- `バグ、要望、コメントの送信` — GitHub の Issue ページを開きます。
- `マニュアル (PDF)` — このオンラインマニュアルを既定のブラウザで開きます。

## 言語

`言語` メニューから表示言語を切り替えられます。切り替えには再起動が必要です（再起動は自動で行われます）。CSManager は英語・日本語のほか、ドイツ語・フランス語・スペイン語・ポルトガル語・イタリア語・ロシア語・中国語（簡体字・繁体字）・韓国語に対応しています。
