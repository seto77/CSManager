# Ricerca

Il pannello di ricerca consente di filtrare il database cristallino caricato in base a nome, composizione, simmetria, costanti reticolari, valore d, densità, informazioni bibliografiche e altro ancora.

## Contenuti previsti

- Ricerca testuale per nome e riferimenti bibliografici.
- Inclusione ed esclusione di elementi tramite la finestra di dialogo della tavola periodica.
- Filtraggio per sistema cristallino e gruppo spaziale.
- Specifica degli intervalli delle costanti reticolari.
- Ricerca per valore d, con o senza considerazione del fattore di diffusione.
- Specifica dell'intervallo di densità.
