# Opzioni e lingua

Questa pagina tratta le opzioni dell'applicazione, le impostazioni di visualizzazione e il cambio di lingua.

## Contenuti previsti

- Suggerimenti (tooltip).
- Caricamento automatico del database AMCSD all'avvio.
- Ricerca incrementale.
- Dimensione del carattere nell'elenco dei cristalli.
- Cambio di lingua tra inglese e giapponese e riavvio.
- Voci del menu Guida.
