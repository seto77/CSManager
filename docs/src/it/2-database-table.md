# Elenco dei cristalli

L'elenco dei cristalli mostra le voci cristalline caricate. Selezionando una riga, l'editor delle informazioni sul cristallo nella parte inferiore viene aggiornato.

## Contenuti previsti

- Ordinamento delle colonne.
- Commutazione della visualizzazione tra AMCSD e COD.
- Lettura delle colonne relative a formula chimica, densità, costanti reticolari, simmetria e riferimenti bibliografici.
- Invio del cristallo selezionato alle applicazioni correlate.
