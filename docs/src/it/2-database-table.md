# Elenco dei cristalli

Una volta caricato il database, nella parte centrale della finestra principale viene visualizzato l'elenco dei cristalli. Ogni riga corrisponde a un cristallo e le colonne mostrano informazioni quali nome, composizione chimica, simmetria, parametri di cella, densità e riferimenti bibliografici.

## Selezione

Selezionando un cristallo, i suoi dettagli vengono visualizzati nelle [Informazioni sul cristallo](4-crystal-information/index.md) nella parte inferiore. Inoltre, se PDIndexer o ReciPro sono in esecuzione, all'atto della selezione le informazioni del cristallo vengono trasferite a questi software tramite gli appunti (vedere [Interoperabilità con altri software](6-interoperation.md)).

## Ordinamento

Facendo clic sull'intestazione di una colonna (parte superiore), è possibile ordinare le voci in ordine crescente o decrescente in base a quel campo.
