# Manuale di CSManager

CSManager è un'applicazione desktop per Windows che consente di consultare, cercare e modificare database di strutture cristalline. Supporta il caricamento del database AMCSD incluso, l'acquisizione su richiesta di COD, l'importazione di file cristallini, la modifica delle informazioni sui cristalli e lo scambio di dati con i software correlati.

Questa versione del manuale, pubblicata su GitHub Pages, sarà gestita come fonte di riferimento ufficiale d'ora in poi. Le pagine iniziali forniscono una struttura di base ricavata dall'impostazione del PDF/manuale esistente e dall'interfaccia attuale; i contenuti dettagliati verranno ampliati direttamente nel Markdown.
