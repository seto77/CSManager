# Manuale di CSManager

CSManager è un'applicazione desktop per Windows che consente di cercare, consultare, modificare ed esportare le strutture cristalline fornite dall'American Mineralogist Crystal Structure Database (AMCSD) e dal Crystallography Open Database (COD). Include il database AMCSD con circa 20.000 voci ed è inoltre in grado di scaricare su richiesta il database COD, che supera le 500.000 voci. I cristalli caricati possono essere importati ed esportati nei formati CIF / AMC e il cristallo selezionato può essere trasferito a software correlati come PDIndexer o ReciPro per l'interoperabilità.
