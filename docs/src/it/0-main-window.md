# Finestra principale

La finestra principale è organizzata attorno all'elenco dei cristalli, alle informazioni del cristallo selezionato e al pannello di ricerca.

## Aree

- I menu File e dell'applicazione nella parte superiore.
- L'elenco dei cristalli al centro.
- L'editor delle informazioni sul cristallo per consultare e modificare il cristallo selezionato.
- Il pannello di ricerca per filtrare il database caricato.
- L'area di stato che mostra l'avanzamento e i messaggi.

## Note di redazione

- Le etichette dell'interfaccia vanno verificate prendendo come riferimento `CSManager/FormMain.it.resx`.
- Le schermate vanno spostate in `docs/src/assets/cap-it-auto/` oppure in `docs/src/assets/cap-it-manual/`.
