# Importazione ed esportazione

CSManager può importare dati cristallini da file CIF, AMC e XML di PDIndexer/ReciPro. Supporta inoltre l'esportazione in CIF del cristallo selezionato e il salvataggio del database modificato.

## Contenuti previsti

- `Importa CIF, AMC …`
- `Importa dati cristallini PDI (*.xml)`
- Trascinamento (drag-and-drop) nell'area delle informazioni sul cristallo.
- Salvataggio del database.
- Esportazione del cristallo selezionato.
