# Importazione ed esportazione

CSManager può importare cristalli dai formati CIF, AMC e dagli elenchi di cristalli di PDIndexer / ReciPro, ed esportare il cristallo selezionato come CIF.

## Importazione

- `Importa CIF, AMC …` nel menu `File` — legge un file CIF o AMC e lo aggiunge in coda al database corrente.
- `Importa dati cristallini PDI (*.xml)` nel menu `File` — legge un elenco di cristalli di PDIndexer o ReciPro (`*.xml`) e lo aggiunge in coda al database corrente.
- È inoltre possibile caricare un cristallo trascinando un file CIF / AMC nell'area [Informazioni sul cristallo](4-crystal-information/index.md). In alternativa, è possibile importare facendo clic con il tasto destro sulle informazioni del cristallo e scegliendo `Importa da CIF o AMC...`.

CIF è il formato standard dell'Unione Internazionale di Cristallografia (IUCr), mentre AMC è il formato adottato dalla Mineralogical Society of America.

## Esportazione

- Facendo clic con il tasto destro sulle [Informazioni sul cristallo](4-crystal-information/index.md) e scegliendo `Esporta in CIF`, è possibile esportare il cristallo corrente in formato CIF.
- Per salvare l'intero database, utilizzare `Salva database` nel menu `File` (vedere [Caricamento del database](1-database-loading.md)).
