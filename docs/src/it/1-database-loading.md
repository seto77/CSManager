# Caricamento del database

Il caricamento e il salvataggio del database si effettuano dal menu `File`.

## Database standard

- `Leggi database AMCSD (### cristalli)` — carica il database AMCSD fornito dalla Mineralogical Society of America (MSA). Contiene circa 20.000 strutture cristalline ed è incluso nel file di installazione, quindi non è necessario alcun download separato. Il menu indica il numero di voci attualmente disponibili. Se in `Opzioni` è attivata l'opzione «Leggi database AMCSD al prossimo avvio», il database viene caricato automaticamente all'avvio.
- `Leggi database COD (### cristalli)` — carica il database cristallino fornito dal Crystallography Open Database (COD). Contiene oltre 500.000 strutture cristalline e, a causa delle dimensioni elevate del file (oltre 800 MB), non è incluso. Al primo caricamento viene scaricato automaticamente da Internet; nei caricamenti successivi viene letto il file già scaricato.

## Database personalizzati

- `Leggi database` — apre un qualsiasi file di database CSManager (`*.cdb3`).
- `Salva database` — salva il database corrente in formato `*.cdb3`.
- `Svuota database` — cancella i dati già caricati. Il file di database in sé non viene eliminato.

Se si utilizza direttamente AMCSD o COD, raramente si avrà bisogno di usare `Leggi database` / `Salva database`.

## Utilizzo e citazione del database

Quando si pubblicano articoli accademici o simili utilizzando i dati delle strutture cristalline contenuti in AMCSD, sia online sia offline, è obbligatorio citare il seguente articolo.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
