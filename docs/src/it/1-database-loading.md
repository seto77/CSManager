# Caricamento del database

CSManager può caricare il database AMCSD incluso e, se necessario, il database COD acquisito su richiesta.

## Database standard

- `Leggi database AMCSD (### cristalli)` carica il set di dati AMCSD incluso.
- `Leggi database COD (### cristalli)` carica i dati COD. Al primo utilizzo può essere necessario scaricare il grande archivio COD.

## Database personalizzati

- `Leggi database` apre un file di database CSManager esistente.
- `Salva database` salva il database corrente.
- `Svuota database` cancella le voci caricate. Il file di database vero e proprio non viene eliminato.
