# Interoperabilità con altri software

CSManager può scambiare dati cristallini con strumenti correlati come PDIndexer e ReciPro.

## Contenuti previsti

- Trasferimento tramite gli appunti quando un cristallo è selezionato.
- Importazione dell'elenco dei cristalli di PDIndexer / ReciPro.
- Formati di file e di dati previsti.
