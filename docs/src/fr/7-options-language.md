# Options et langue

## Options

Le menu `Options` propose les réglages suivants.

- `Info-bulle` — lorsqu'il est activé, affiche une info-bulle (bulle explicative) sur chaque contrôle.
- `Lire la base AMCSD au prochain démarrage` — lorsqu'il est activé, charge la base AMCSD dès le lancement. Activé par défaut.
- `Recherche incrémentale` — lorsqu'elle est activée, répercute immédiatement les modifications des critères de recherche sur les résultats.
- `Taille de police du tableau de données` — modifie la taille de police de la liste des cristaux.

## Aide

Le menu `Aide` comporte les éléments suivants.

- `Mises à jour du programme` — vérifie si une nouvelle version est disponible et, le cas échéant, effectue la mise à jour.
- `Historique des versions` — affiche l'historique des versions précédentes.
- `Licence` — affiche la licence.
- `Page Github` — ouvre le dépôt CSManager sur GitHub.
- `Signaler des bugs, demandes ou commentaires` — ouvre la page des Issues GitHub.
- `Aide (PDF)` — ouvre ce manuel en ligne dans le navigateur par défaut.

## Langue

Le menu `Langue` permet de changer la langue d'affichage. Le changement nécessite un redémarrage (qui s'effectue automatiquement). Outre l'anglais et le japonais, CSManager prend en charge l'allemand, le français, l'espagnol, le portugais, l'italien, le russe, le chinois (simplifié et traditionnel) et le coréen.
