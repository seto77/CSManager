# Dépannage

Cette page rassemble les problèmes courants et les étapes de résolution.

## Contenu à venir

- Échecs de téléchargement ou de chargement de COD.
- Absence du .NET Desktop Runtime.
- Fichiers de base de données impossibles à ouvrir.
- Recherche lente ou comportement des grandes bases de données.
- Problèmes d'échange de données avec PDIndexer ou ReciPro.
