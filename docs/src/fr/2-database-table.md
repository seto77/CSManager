# Liste des cristaux

Une fois la base de données chargée, la liste des cristaux s'affiche dans la partie centrale de la fenêtre principale. Chaque ligne correspond à un cristal, et des informations telles que le nom, la composition chimique, la symétrie, les paramètres de maille, la densité et les références sont affichées dans les colonnes.

## Sélection

Lorsque vous sélectionnez un cristal, ses détails s'affichent dans les [Informations cristallines](4-crystal-information/index.md) en bas. Par ailleurs, si PDIndexer ou ReciPro est en cours d'exécution, les informations de ce cristal sont transférées vers ces logiciels via le presse-papiers au moment de la sélection (voir [Interopérabilité](6-interoperation.md)).

## Tri

En cliquant sur l'en-tête d'une colonne (tout en haut), vous pouvez trier les cristaux par ordre croissant ou décroissant selon cet élément.
