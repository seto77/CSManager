# Liste des cristaux

La liste des cristaux affiche les entrées cristallines chargées. La sélection d'une ligne met à jour l'éditeur d'informations cristallines.

## Contenu à venir

- Tri des colonnes.
- Filtrage des entrées AMCSD et COD.
- Lecture des colonnes de formule, de densité, de paramètres de maille, de symétrie et de références.
- Envoi du cristal sélectionné vers les applications associées.
