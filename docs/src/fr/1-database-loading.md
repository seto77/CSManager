# Chargement de la base de données

CSManager peut charger la base AMCSD fournie ainsi que la base COD facultative.

## Bases de données intégrées

- `Lire la base AMCSD (### cristaux)` charge le jeu de données AMCSD fourni.
- `Lire la base COD (### cristaux)` charge les données COD. La première utilisation peut nécessiter le téléchargement de la volumineuse archive COD.

## Bases de données personnalisées

- `Lire la base de données` ouvre un fichier de base de données CSManager existant.
- `Enregistrer la base de données` enregistre la base de données actuelle.
- `Effacer la base de données` efface les entrées chargées sans supprimer les fichiers de base de données.
