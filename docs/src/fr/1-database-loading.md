# Chargement de la base de données

Le chargement et l'enregistrement des bases de données s'effectuent depuis le menu `Fichier`.

## Bases de données standard

- `Lire la base AMCSD (### cristaux)` — charge la base AMCSD fournie par la Mineralogical Society of America (MSA). Elle contient environ 20 000 structures cristallines et, étant incluse dans le fichier d'installation, aucun téléchargement séparé n'est nécessaire. Le menu indique le nombre d'entrées actuellement disponibles. Si l'option « Lire la base AMCSD au prochain démarrage » est activée dans `Options`, elle est chargée automatiquement au lancement.
- `Lire la base COD (### cristaux)` — charge la base cristalline fournie par la Crystallography Open Database (COD). Elle contient plus de 500 000 structures cristallines et n'est pas incluse en raison de la taille importante du fichier (plus de 800 Mo). Lors du premier chargement, elle est téléchargée automatiquement depuis Internet ; lors des chargements suivants, le fichier déjà téléchargé est utilisé.

## Bases de données personnalisées

- `Lire la base de données` — ouvre n'importe quel fichier de base de données CSManager (`*.cdb3`).
- `Enregistrer la base de données` — enregistre la base de données actuelle au format `*.cdb3`.
- `Effacer la base de données` — efface les données déjà chargées. Le fichier de base de données lui-même n'est pas supprimé.

Si vous utilisez AMCSD ou COD telles quelles, vous aurez rarement l'occasion d'utiliser `Lire la base de données` / `Enregistrer la base de données`.

## Utilisation et citation des bases de données

Si vous publiez des articles scientifiques ou autres en utilisant les données de structures cristallines contenues dans AMCSD, vous devez impérativement citer l'article suivant, que ce soit en ligne ou hors ligne.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
