# Interopérabilité

CSManager peut échanger des données cristallines avec des logiciels associés tels que PDIndexer et ReciPro.

## Transfert automatique à la sélection

Lorsque vous sélectionnez un cristal dans la [Liste des cristaux](2-database-table.md), si PDIndexer ou ReciPro est en cours d'exécution, les informations de ce cristal sont transférées automatiquement via le presse-papiers. Il suffit de sélectionner un cristal dans CSManager pour charger le même cristal dans le logiciel cible.

Vous pouvez également envoyer le cristal actuel en faisant un clic droit sur les [Informations cristallines](4-crystal-information/index.md) et en sélectionnant `Send this crystal to other software`.

## Importation de listes de cristaux

Depuis `Importer des données cristallines PDI (*.xml)` du menu `Fichier`, vous pouvez charger une liste de cristaux PDIndexer ou ReciPro (`*.xml`) (voir [Importation et exportation](5-import-export.md)).
