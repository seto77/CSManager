# Interopérabilité

CSManager peut échanger des données cristallines avec des outils associés tels que PDIndexer et ReciPro.

## Contenu à venir

- Transfert via le presse-papiers lorsqu'un cristal est sélectionné.
- Importation de listes de cristaux depuis PDIndexer ou ReciPro.
- Formats de fichiers et de données attendus.
