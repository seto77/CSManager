# Importation et exportation

CSManager peut importer des données cristallines à partir de fichiers CIF, AMC et XML PDIndexer/ReciPro. Il peut exporter le cristal sélectionné au format CIF et enregistrer les bases de données organisées sous forme de fichiers de base de données CSManager.

## Contenu à venir

- `Importer CIF, AMC …`
- `Importer des données cristallines PDI (*.xml)`
- Importation par glisser-déposer dans la zone d'informations cristallines.
- Enregistrement des bases de données.
- Exportation des cristaux sélectionnés.
