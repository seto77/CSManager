# Références

L'onglet `Réf.` affiche et permet de configurer les informations bibliographiques constituant la source des données de structure cristalline.

- `Note` — saisissez des notes ou des mémos.
- `Authors` — les noms des auteurs de l'article cité.
- `Journal` — le nom de la revue de l'article cité.
- `Title` — le titre de l'article cité.
