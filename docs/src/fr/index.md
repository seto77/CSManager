# Manuel de CSManager

CSManager est une application de bureau Windows permettant de consulter, rechercher et maintenir des bases de données de structures cristallines. Elle peut charger la base AMCSD fournie, télécharger et charger COD à la demande, importer les formats cristallins courants, modifier les informations cristallines et échanger des données cristallines sélectionnées avec des logiciels associés.

Ce manuel hébergé sur GitHub Pages constitue désormais la documentation de référence. Les pages initiales sont une ébauche fondée sur la structure du PDF/manuel existant et sur l'interface actuelle de l'application ; le contenu détaillé sera développé directement dans le Markdown.
