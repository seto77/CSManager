# Manuel de CSManager

CSManager est une application de bureau Windows permettant de rechercher, consulter, modifier et exporter les structures cristallines fournies par l'American Mineralogist Crystal Structure Database (AMCSD) et la Crystallography Open Database (COD). Il intègre la base AMCSD comptant environ 20 000 entrées et permet en outre de récupérer à la demande la base COD, qui dépasse les 500 000 entrées. Les cristaux chargés peuvent être importés et exportés aux formats CIF / AMC, et le cristal sélectionné peut être transféré vers des logiciels associés tels que PDIndexer ou ReciPro pour assurer l'interopérabilité.
