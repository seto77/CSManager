# Datenbank laden

Das Laden und Speichern von Datenbanken erfolgt über das Menü `Datei`.

## Standarddatenbanken

- `AMCSD-Datenbank laden (### Kristalle)` — lädt die von der Mineralogical Society of America (MSA) bereitgestellte AMCSD-Datenbank. Sie enthält rund 20.000 Kristallstrukturen und ist in der Installationsdatei enthalten, sodass kein separater Download erforderlich ist. Im Menü wird die aktuelle Anzahl der Einträge angezeigt. Wenn Sie unter `Optionen` „AMCSD-Datenbank beim nächsten Start laden“ aktiviert haben, wird sie beim Start automatisch geladen.
- `COD-Datenbank laden (### Kristalle)` — lädt die von der Crystallography Open Database (COD) bereitgestellte Kristalldatenbank. Sie enthält mehr als 500.000 Kristallstrukturen und ist wegen ihrer großen Dateigröße (über 800 MB) nicht enthalten. Beim ersten Laden wird sie automatisch aus dem Internet heruntergeladen; bei späteren Aufrufen wird die bereits heruntergeladene Datei geladen.

## Benutzerdefinierte Datenbanken

- `Datenbank laden` — öffnet eine beliebige CSManager-Datenbankdatei (`*.cdb3`).
- `Datenbank speichern` — speichert die aktuelle Datenbank im Format `*.cdb3`.
- `Datenbank leeren` — löscht die geladenen Daten. Die Datenbankdatei selbst wird dabei nicht gelöscht.

Wenn Sie AMCSD oder COD unverändert verwenden, werden Sie `Datenbank laden` / `Datenbank speichern` nur selten benötigen.

## Nutzung und Zitierung der Datenbank

Wenn Sie Kristallstrukturdaten aus der AMCSD für wissenschaftliche Veröffentlichungen oder Ähnliches verwenden, müssen Sie – ob online oder offline – stets die folgende Arbeit zitieren.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
