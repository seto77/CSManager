# Datenbank laden

CSManager kann die mitgelieferte AMCSD-Datenbank sowie die bei Bedarf abgerufene COD-Datenbank laden.

## Integrierte Datenbanken

- `AMCSD-Datenbank laden (### Kristalle)` lädt den mitgelieferten AMCSD-Datensatz.
- `COD-Datenbank laden (### Kristalle)` lädt COD-Daten. Beim ersten Mal kann das Herunterladen des großen COD-Archivs erforderlich sein.

## Benutzerdefinierte Datenbanken

- `Datenbank laden` öffnet eine vorhandene CSManager-Datenbankdatei.
- `Datenbank speichern` speichert die aktuelle Datenbank.
- `Datenbank leeren` löscht die geladenen Einträge, ohne die Datenbankdateien selbst zu löschen.
