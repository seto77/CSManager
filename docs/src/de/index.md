# CSManager-Handbuch

CSManager ist eine Windows-Desktop-Anwendung zum Suchen, Anzeigen, Bearbeiten und Exportieren von Kristallstrukturen, die von der American Mineralogist Crystal Structure Database (AMCSD) und der Crystallography Open Database (COD) bereitgestellt werden. Die AMCSD-Datenbank mit rund 20.000 Einträgen ist bereits enthalten, und die mehr als 500.000 Einträge umfassende COD-Datenbank kann bei Bedarf abgerufen werden. Geladene Kristalle lassen sich im CIF- oder AMC-Format importieren und exportieren, und ausgewählte Kristalle können an verwandte Programme wie PDIndexer oder ReciPro übertragen werden.
