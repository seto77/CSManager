# Kristallliste

Wenn Sie eine Datenbank laden, wird in der Mitte des Hauptfensters eine Übersicht der Kristalle (die Kristallliste) angezeigt. Jede Zeile entspricht einem Kristall, und in den Spalten werden Informationen wie Name, chemische Zusammensetzung, Symmetrie, Gitterkonstanten, Dichte und Literatur angezeigt.

## Auswahl

Wenn Sie einen Kristall auswählen, werden seine Details unten in den [Kristallinformationen](4-crystal-information/index.md) angezeigt. Sind außerdem PDIndexer oder ReciPro gestartet, werden die Informationen des Kristalls bei der Auswahl gleichzeitig über die Zwischenablage an diese Programme übertragen (siehe [Zusammenarbeit mit anderer Software](6-interoperation.md)).

## Sortieren

Wenn Sie auf die Überschrift einer Spalte (ganz oben) klicken, wird die Liste nach diesem Eintrag auf- oder absteigend sortiert.
