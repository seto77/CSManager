# Kristallliste

In der Kristallliste werden die geladenen Kristalleinträge angezeigt. Beim Auswählen einer Zeile wird der darunter liegende Kristallinformations-Editor aktualisiert.

## Geplante Inhalte

- Spalten sortieren.
- Anzeige von AMCSD- und COD-Einträgen umschalten.
- Lesen der Spalten Summenformel, Dichte, Gitterkonstanten, Symmetrie und Literatur.
- Senden des ausgewählten Kristalls an verwandte Anwendungen.
