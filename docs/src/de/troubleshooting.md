# Fehlerbehebung

Diese Seite fasst häufige Probleme und Lösungsschritte zusammen.

## Geplante Inhalte

- Fehler beim Herunterladen oder Laden von COD.
- Fehlende .NET Desktop Runtime.
- Datenbankdateien, die sich nicht öffnen lassen.
- Langsame Suche bei großen Datenbanken.
- Probleme beim Datenaustausch mit PDIndexer oder ReciPro.
