# Import und Export

CSManager kann Kristalle aus CIF, AMC sowie aus den Kristalllisten von PDIndexer / ReciPro importieren und den ausgewählten Kristall als CIF exportieren.

## Import

- `CIF, AMC … importieren` im Menü `Datei` — lädt eine CIF- oder AMC-Datei und fügt sie am Ende der aktuellen Datenbank an.
- `PDI-Kristalldaten importieren (*.xml)` im Menü `Datei` — lädt eine Kristallliste von PDIndexer oder ReciPro (`*.xml`) und fügt sie am Ende der aktuellen Datenbank an.
- Sie können Kristalle auch laden, indem Sie eine CIF- oder AMC-Datei per Drag & Drop in den Bereich [Kristallinformationen](4-crystal-information/index.md) ziehen. Ebenso können Sie mit der rechten Maustaste auf die Kristallinformationen klicken und `Aus CIF oder AMC importieren...` wählen.

CIF ist das Standardformat der International Union of Crystallography (IUCr), AMC ist das von der Mineralogical Society of America verwendete Format.

## Export

- Wenn Sie mit der rechten Maustaste auf die [Kristallinformationen](4-crystal-information/index.md) klicken und `Als CIF exportieren` wählen, können Sie den aktuellen Kristall im CIF-Format exportieren.
- Um die gesamte Datenbank zu speichern, verwenden Sie `Datenbank speichern` im Menü `Datei` (siehe [Datenbank laden](1-database-loading.md)).
