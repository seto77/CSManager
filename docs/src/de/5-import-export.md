# Import und Export

CSManager kann Kristalldaten aus CIF-, AMC- und PDIndexer/ReciPro-XML-Dateien importieren. Außerdem lässt sich der ausgewählte Kristall als CIF exportieren und die bearbeitete Datenbank speichern.

## Geplante Inhalte

- `CIF, AMC … importieren`
- `PDI-Kristalldaten importieren (*.xml)`
- Drag-and-Drop-Import in den Bereich Kristallinformationen.
- Speichern von Datenbanken.
- Exportieren ausgewählter Kristalle.
