# Optionen und Sprache

## Optionen

Das Menü `Optionen` enthält die folgenden Einstellungen.

- `Tooltip` — wenn aktiviert, werden für die einzelnen Steuerelemente Tooltips (Erklärungsblasen) angezeigt.
- `AMCSD-Datenbank beim nächsten Start laden` — wenn aktiviert, wird die AMCSD-Datenbank beim Start automatisch geladen. Standardmäßig aktiviert.
- `Inkrementelle Suche` — wenn aktiviert, werden Änderungen an den Suchkriterien sofort in den Suchergebnissen berücksichtigt.
- `Schriftgröße in der Datentabelle` — ändert die Schriftgröße in der Kristallliste.

## Hilfe

Das Menü `Hilfe` enthält die folgenden Einträge.

- `Programm-Updates` — prüft, ob eine neue Version veröffentlicht wurde, und führt gegebenenfalls ein Update durch.
- `Versionsverlauf` — zeigt den bisherigen Versionsverlauf an.
- `Lizenz` — zeigt die Lizenz an.
- `Github-Seite` — öffnet das CSManager-Repository auf GitHub.
- `Fehler, Wünsche oder Kommentare melden` — öffnet die Issue-Seite auf GitHub.
- `Hilfe (PDF)` — öffnet dieses Online-Handbuch im Standardbrowser.

## Sprache

Über das Menü `Sprache` können Sie die Anzeigesprache wechseln. Für den Wechsel ist ein Neustart erforderlich (der Neustart erfolgt automatisch). CSManager unterstützt neben Englisch und Japanisch auch Deutsch, Französisch, Spanisch, Portugiesisch, Italienisch, Russisch, Chinesisch (vereinfacht und traditionell) und Koreanisch.
