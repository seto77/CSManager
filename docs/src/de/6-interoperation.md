# Zusammenarbeit mit anderer Software

CSManager kann Kristalldaten mit verwandten Programmen wie PDIndexer und ReciPro austauschen.

## Automatische Übertragung bei der Auswahl

Wenn Sie in der [Kristallliste](2-database-table.md) einen Kristall auswählen und PDIndexer oder ReciPro gestartet sind, werden die Informationen dieses Kristalls automatisch über die Zwischenablage übertragen. Sie müssen lediglich in CSManager einen Kristall auswählen, um denselben Kristall in das Zielprogramm zu laden.

Sie können den aktuellen Kristall auch senden, indem Sie mit der rechten Maustaste auf die [Kristallinformationen](4-crystal-information/index.md) klicken und `Send this crystal to other software` wählen.

## Import von Kristalllisten

Über `PDI-Kristalldaten importieren (*.xml)` im Menü `Datei` können Sie eine Kristallliste von PDIndexer oder ReciPro (`*.xml`) laden (siehe [Import und Export](5-import-export.md)).
