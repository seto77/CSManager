# Zusammenarbeit mit anderer Software

CSManager kann Kristalldaten mit verwandten Werkzeugen wie PDIndexer und ReciPro austauschen.

## Geplante Inhalte

- Übertragung über die Zwischenablage bei ausgewähltem Kristall.
- Importieren von Kristalllisten aus PDIndexer oder ReciPro.
- Erwartete Datei- und Datenformate.
