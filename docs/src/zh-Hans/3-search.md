# 检索

在检索面板中，可按各类条件筛选已读入的晶体数据库。勾选要使用条件的复选框并输入数值，然后按 `搜索` 按钮或按 Enter 键即可执行检索。可同时指定多个条件。

## 检索条件

- `Name` — 按晶体的名称检索。
- `元素` — 按 `周期表` 按钮会打开另一个窗口，可在其中选择检索对象的元素。各元素按钮每按一次便切换一次状态（可含可不含 / 必须包含 / 必须排除）。通过窗口上部的 "may or not include" / "must include" / "must exclude" 按钮，可一次性切换全部元素的状态。
- `参考文献` — 按论文名、期刊名、作者名等文献信息检索。
- `晶系` — 按晶系筛选。
- `晶胞参数` — 指定晶胞参数及其容许范围。容许范围输入为零的项将被忽略。
- `晶面间距` — 指定晶面的 d 值（晶面间距）及其容许范围，检索其 d 值落入该范围内的晶体。启用 `忽略散射因子` 时，将不考虑衍射强度（晶体结构因子），并把违反消光规律的晶面也纳入检索。未启用时则考虑晶体结构因子，仅以强度排名前 20 的晶面作为检索对象。
- `密度` — 指定密度及其容许范围。
