# 主窗口

启动 CSManager 后会显示主窗口。窗口大致由以下几个区域组成。

- **菜单（最上部）** — 包含 `文件`、`选项`、`帮助`、`语言` 各菜单。用于读写数据库和设置程序。
- **晶体列表（中央部）** — 已读入晶体的一览。选中某行后，下方的晶体信息会随之更新。
- **晶体信息（下部）** — 显示和编辑选中晶体的晶胞参数、对称性、原子位置等。
- **检索面板（下部右侧）** — 按各类条件筛选已读入的数据库。
- **状态区域（最下部）** — 显示读取进度和处理结果的消息。
