# 数据库读取

数据库的读取与保存通过 `文件` 菜单进行。

## 标准数据库

- `读取 AMCSD 数据库（含 ### 个晶体）` — 读取由美国矿物学会 (MSA) 提供的 AMCSD 数据库。该数据库收录约 2 万个晶体结构，已随安装文件一并附带，无需另行下载。菜单中会显示当前收录的数量。若在 `选项` 中启用了「下次启动时读取 AMCSD 数据库」，则会在启动时自动读取。
- `读取 COD 数据库（含 ### 个晶体）` — 读取由 Crystallography Open Database (COD) 提供的晶体数据库。该数据库收录超过 50 万个晶体结构，由于文件体积庞大（超过 800 MB）而未随程序附带。首次读取时会自动从互联网下载，此后将读取已下载的文件。

## 自定义数据库

- `读取数据库` — 打开任意的 CSManager 数据库文件 (`*.cdb3`)。
- `保存数据库` — 将当前数据库以 `*.cdb3` 格式保存。
- `清空数据库` — 清除已读入的数据。数据库文件本身不会被删除。

若直接使用 AMCSD 或 COD，几乎不会用到 `读取数据库` / `保存数据库`。

## 数据库的使用与引用

若使用 AMCSD 中包含的晶体结构数据出版学术论文等，无论在线还是离线，都务必引用以下论文。

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
