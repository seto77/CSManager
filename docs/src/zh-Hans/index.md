# CSManager 手册

CSManager 是一款用于浏览、检索和编辑晶体结构数据库的 Windows 桌面应用程序。它支持读取捆绑的 AMCSD 数据库、按需获取 COD、导入晶体文件、编辑晶体信息，以及与相关软件之间的数据互通。

本 GitHub Pages 版手册将作为今后的正式文档来源进行维护。初始页面依据现有 PDF/手册的结构和当前 UI 搭建框架，详细正文将直接在 Markdown 中补充。
