# 选项与语言

## 选项

`选项` 菜单包含以下设置。

- `工具提示` — 启用后，会在各控件上显示工具提示（说明气泡）。
- `下次启动时读取 AMCSD 数据库` — 启用后，会在启动的同时读取 AMCSD 数据库。默认启用。
- `增量搜索` — 启用后，在更改检索条件的瞬间即会立刻反映到检索结果中。
- `数据表字体大小` — 更改晶体列表的字号。

## 帮助

`帮助` 菜单包含以下项目。

- `程序更新` — 检查是否已发布新版本，若有则进行更新。
- `版本历史` — 显示历来的版本历史。
- `许可证` — 显示许可证。
- `Github 页面` — 打开 GitHub 上的 CSManager 仓库。
- `报告错误、需求或意见` — 打开 GitHub 的 Issue 页面。
- `帮助 (PDF)` — 在默认浏览器中打开本在线手册。

## 语言

可通过 `语言` 菜单切换显示语言。切换需要重启（重启会自动进行）。CSManager 除英语、日语外，还支持德语、法语、西班牙语、葡萄牙语、意大利语、俄语、中文（简体、繁体）以及韩语。
