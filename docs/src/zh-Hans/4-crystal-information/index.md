# 晶体信息

在晶体信息区域（`Crystal Information` 控件）中，可显示和编辑选中晶体的晶胞参数、对称性、原子位置、文献等。也可将 CIF 文件或 AMC 文件拖放到该区域，以读入任意晶体。

!!! warning "更改需用 `添加` / `替换` 确认"
    对晶体进行更改后，请务必按 `添加` 或 `替换` 按钮。若未按下，更改内容将不会保存到晶体列表中而丢失。

最上部显示晶体名 (`Name`) 和化学组成 (`Formula`) 等共通信息，可用 `Reset` 将晶体的全部信息初始化。下部排列着用于设置对称性、原子、文献等的标签页。

## 标签页

- [基本信息](1-basic-information.md) — 晶胞参数、对称性、体积、密度等（`基本信息` 标签页）
- [原子](2-atoms.md) — 原子位置、占有率、温度因子、散射因子（`原子信息` 标签页）
- [键与配位多面体](3-bonds-polyhedra.md) — 键与配位多面体的绘制设置（`键与多面体` 标签页）
- [文献](4-references.md) — 引用文献信息（`文献` 标签页）

## 右键菜单

右键点击控件的空白部分，可进行以下操作。

- `从 CIF 或 AMC 导入...` — 以 CIF / AMC 格式导入晶体结构。
- `导出为 CIF` — 将当前晶体以 CIF 格式导出。
- `Send this crystal to other software` — 将选中的晶体发送至 PDIndexer、ReciPro 等。
- `对称性信息` — 打开与对称性相关的详细窗口（空间群通用编号、劳厄群与点群、出现规律、Wyckoff position、晶面与晶带轴的几何计算等）。
- `还原晶胞常数` — 将晶胞参数还原为刚读入时的数值。
- `转换为 P1 空间群` / `转换为超结构` / `更改坐标轴/原点设置` — 执行空间群或晶格的变换。
