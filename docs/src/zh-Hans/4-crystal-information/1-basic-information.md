# 基本信息

在 `基本信息` 标签页中，显示和设置晶体的晶胞参数与对称性。

## 晶胞参数与对称性

- **Cell constants** — 显示和设置晶胞参数。长度单位为 Å (10⁻¹⁰ m)。指定对称性后若晶胞参数存在约束（例如立方晶系的 a = b = c, α = β = γ = 90°），将自动重新设定。
- **Symmetry** — 按层级设置对称性。指定 `Crystal system`（晶系）、`Point group`（点群）、`Space group`（空间群）。空间群请注意坐标轴的设置。在 `Search` 中输入空间群字符串后，候选会显示在右侧列表中（区分大小写）。

## 计算得到的值

在已输入原子信息的情况下，会计算并显示以下数值。

- **Cell Volume** — 单位晶胞的体积。
- **Cell Mass** — 单位晶胞的质量。
- **Molar Volume** / **Molar Mass** — 每摩尔的体积与质量。
- **Z** — 单位晶胞中包含的化学式单元数。
- **Density** — 密度。
- **Color of Profile** — 绘制衍射峰所用的颜色。点击后会打开颜色设置窗口。

更详细的对称性信息（劳厄群与点群的各类表记、出现规律、Wyckoff position、晶面与晶带轴的几何计算）可通过右键菜单的 `对称性信息` 查看。
