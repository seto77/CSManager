# 键与配位多面体

在 `键与多面体` 标签页中，设置以三维方式显示晶体结构时键与配位多面体的绘制。此处定义的键与多面体，将用于 ReciPro 等结构查看器中的显示。

## 键的定义

- `成键原子` — 选择要成键的两类原子（中心原子与顶点原子）。
- `添加` / `替换` / `删除` — 将设置好的键添加到列表、替换当前选中的键，或将其删除。

## 显示选项

- `显示键` — 显示原子间的键。
- `显示多面体` — 显示配位多面体。
- `显示棱边` — 显示多面体的棱（边）。
- `显示顶点原子` / `显示中心原子` — 显示多面体的顶点原子与中心原子。
- `显示内部键` — 显示多面体内部的键。

多面体的透明度等细致的绘制设置，也可在此标签页中调整。
