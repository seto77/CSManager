# 与其他软件的协同

CSManager 可与 PDIndexer、ReciPro 等相关软件交换晶体数据。

## 选中时的自动转送

在 [晶体列表](2-database-table.md) 中选中某个晶体时，若 PDIndexer 或 ReciPro 正在运行，则该晶体的信息会经由剪贴板自动转送。只需在 CSManager 中选择晶体，即可将同一个晶体读入目标软件。

右键点击 [晶体信息](4-crystal-information/index.md) 并选择 `Send this crystal to other software`，也可发送当前晶体。

## 晶体列表的导入

通过 `文件` 菜单的 `导入 PDI 晶体数据 (*.xml)`，可读取 PDIndexer 或 ReciPro 的晶体列表 (`*.xml`)（参见 [导入与导出](5-import-export.md)）。
