# Manual do CSManager

O CSManager é um aplicativo de desktop para Windows que permite pesquisar, visualizar, editar e exportar as estruturas cristalinas fornecidas pelo American Mineralogist Crystal Structure Database (AMCSD) e pelo Crystallography Open Database (COD). Ele acompanha o banco de dados AMCSD, com cerca de 20 mil registros, e também permite obter sob demanda o banco de dados COD, que ultrapassa 500 mil registros. Os cristais carregados podem ser importados e exportados nos formatos CIF / AMC, e o cristal selecionado pode ser transferido para softwares relacionados como o PDIndexer ou o ReciPro para trabalho integrado.
