# Manual do CSManager

O CSManager é um aplicativo de desktop para Windows que permite navegar, pesquisar e editar bancos de dados de estruturas cristalinas. Ele suporta o carregamento do banco de dados AMCSD incluído, a obtenção sob demanda do COD, a importação de arquivos de cristais, a edição de informações de cristais e a troca de dados com softwares relacionados.

Esta versão do manual em GitHub Pages é mantida como a referência oficial daqui em diante. As páginas iniciais fornecem um esqueleto baseado na estrutura do PDF/manual existente e na interface atual; o conteúdo detalhado será acrescentado diretamente no Markdown.
