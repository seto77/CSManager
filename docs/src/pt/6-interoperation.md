# Interação com outros softwares

O CSManager pode trocar dados de cristais com softwares relacionados, como o PDIndexer e o ReciPro.

## Transferência automática na seleção

Ao selecionar um cristal na [Lista de cristais](2-database-table.md), se o PDIndexer ou o ReciPro estiverem em execução, as informações desse cristal são transferidas automaticamente por meio da área de transferência. Basta selecionar um cristal no CSManager para carregar o mesmo cristal no software de destino.

Também é possível enviar o cristal atual clicando com o botão direito nas [Informações do cristal](4-crystal-information/index.md) e escolhendo `Send this crystal to other software`.

## Importação de listas de cristais

Por meio de `Importar dados de cristais PDI (*.xml)` no menu `Arquivo`, é possível ler uma lista de cristais (`*.xml`) do PDIndexer ou do ReciPro (consulte [Importação e exportação](5-import-export.md)).
