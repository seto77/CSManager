# Interação com outros softwares

O CSManager pode trocar dados de cristais com ferramentas relacionadas, como o PDIndexer e o ReciPro.

## Conteúdo planejado

- Transferência via área de transferência quando um cristal está selecionado.
- Importação de listas de cristais do PDIndexer ou do ReciPro.
- Formatos de arquivo e de dados esperados.
