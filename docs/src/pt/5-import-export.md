# Importação e exportação

O CSManager pode importar dados de cristais a partir de arquivos CIF, AMC e XML do PDIndexer/ReciPro. Também suporta a exportação do cristal selecionado em CIF e o salvamento de bancos de dados editados como arquivos de banco de dados do CSManager.

## Conteúdo planejado

- `Importar CIF, AMC …`
- `Importar dados de cristais PDI (*.xml)`
- Importação por arrastar e soltar na área de informações do cristal.
- Salvamento de bancos de dados.
- Exportação dos cristais selecionados.
