# Importação e exportação

O CSManager pode importar cristais de CIF, AMC e das listas de cristais do PDIndexer / ReciPro, e exportar o cristal selecionado como CIF.

## Importação

- `Importar CIF, AMC …` no menu `Arquivo` — lê um arquivo CIF ou AMC e o adiciona ao final do banco de dados atual.
- `Importar dados de cristais PDI (*.xml)` no menu `Arquivo` — lê uma lista de cristais (`*.xml`) do PDIndexer ou do ReciPro e a adiciona ao final do banco de dados atual.
- Também é possível carregar cristais arrastando e soltando um arquivo CIF / AMC na área de [Informações do cristal](4-crystal-information/index.md). Além disso, a importação pode ser feita clicando com o botão direito nas informações do cristal e escolhendo `Importar de CIF ou AMC...`.

O CIF é o formato padrão da União Internacional de Cristalografia (IUCr), e o AMC é o formato adotado pela Mineralogical Society of America.

## Exportação

- Ao clicar com o botão direito nas [Informações do cristal](4-crystal-information/index.md) e escolher `Exportar para CIF`, é possível exportar o cristal atual no formato CIF.
- Para salvar o banco de dados inteiro, use `Salvar banco de dados` no menu `Arquivo` (consulte [Carregamento do banco de dados](1-database-loading.md)).
