# Janela principal

A janela principal é organizada em torno da lista de cristais, das informações do cristal selecionado e do painel de pesquisa.

## Áreas

- Menus de arquivo e de aplicativo na parte superior.
- Lista de cristais na área central.
- Editor de informações do cristal para conferir e editar o registro selecionado.
- Painel de pesquisa para filtrar o banco de dados carregado.
- Área de status para exibir progresso e mensagens.

## Notas de origem

- Os rótulos da interface devem ser conferidos em `CSManager/FormMain.pt.resx` como referência.
