# Janela principal

Ao iniciar o CSManager, a janela principal é exibida. A janela é dividida, em linhas gerais, nas seguintes áreas.

- **Menu (parte superior)** — os menus `Arquivo`, `Opções`, `Ajuda` e `Idioma`. Permitem ler e gravar bancos de dados e configurar o aplicativo.
- **Lista de cristais (parte central)** — a lista dos cristais carregados. Ao selecionar uma linha, as informações do cristal na parte inferior são atualizadas.
- **Informações do cristal (parte inferior)** — exibe e edita as constantes de célula, a simetria, as posições atômicas e outros dados do cristal selecionado.
- **Painel de pesquisa (parte inferior direita)** — filtra o banco de dados carregado por diversos critérios.
- **Área de status (parte mais inferior)** — exibe mensagens sobre o progresso do carregamento e os resultados do processamento.
