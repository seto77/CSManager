# Solução de problemas

Esta página reúne problemas comuns e formas de resolvê-los.

## Conteúdo planejado

- Falha no download ou no carregamento do COD.
- Quando o .NET Desktop Runtime não é encontrado.
- Quando os arquivos de banco de dados não podem ser abertos.
- Quando a pesquisa fica lenta em bancos de dados grandes.
- Quando a troca de dados com o PDIndexer ou o ReciPro não funciona.
