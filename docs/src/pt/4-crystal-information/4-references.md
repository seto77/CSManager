# Referências

Na aba `Ref.`, exibem-se e configuram-se as informações bibliográficas que são a fonte dos dados de estrutura cristalina.

- `Note` — anota notas ou lembretes.
- `Authors` — os nomes dos autores do artigo citado.
- `Journal` — o nome da revista do artigo citado.
- `Title` — o título do artigo citado.
