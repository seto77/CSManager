# Informações básicas

Esta página trata dos campos básicos do cristal, como nome, fórmula química, parâmetros de rede, volume, densidade, sistema cristalino, grupo pontual e grupo espacial.

## Conteúdo planejado

- Edição dos parâmetros de rede e dos ângulos.
- Atualização dos valores relacionados à fórmula química.
- Conferência das informações de simetria.
- Aplicação das edições com `Adicionar` / `Substituir`.
