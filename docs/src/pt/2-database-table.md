# Lista de cristais

A lista de cristais exibe os registros de cristais carregados. Ao selecionar uma linha, o editor de informações do cristal na parte inferior é atualizado.

## Conteúdo planejado

- Ordenação das colunas.
- Alternância entre a exibição de AMCSD e COD.
- Leitura das colunas de fórmula química, densidade, parâmetros de rede, simetria e referência.
- Envio do cristal selecionado para aplicativos relacionados.
