# Opções e idioma

## Opções

O menu `Opções` contém as seguintes configurações.

- `Dica de ferramenta` — quando ativada, exibe dicas de ferramenta (balões explicativos) em cada controle.
- `Ler banco de dados AMCSD na próxima inicialização` — quando ativada, carrega o banco de dados AMCSD ao iniciar. Ativada por padrão.
- `Pesquisa incremental` — quando ativada, reflete imediatamente os resultados da pesquisa no momento em que um critério de pesquisa é alterado.
- `Tamanho da fonte na tabela de dados` — altera o tamanho da fonte na lista de cristais.

## Ajuda

O menu `Ajuda` contém os seguintes itens.

- `Atualizações do programa` — verifica se há uma nova versão lançada e, se houver, faz a atualização.
- `Histórico de versões` — exibe o histórico de versões até o momento.
- `Licença` — exibe a licença.
- `Página do Github` — abre o repositório do CSManager no GitHub.
- `Relatar erros, solicitações ou comentários` — abre a página de Issues no GitHub.
- `Ajuda (PDF)` — abre este manual on-line no navegador padrão.

## Idioma

Pelo menu `Idioma`, é possível alternar o idioma de exibição. A troca requer reinicialização (que é feita automaticamente). Além de inglês e japonês, o CSManager oferece suporte a alemão, francês, espanhol, português, italiano, russo, chinês (simplificado e tradicional) e coreano.
