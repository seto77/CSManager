# Carregamento do banco de dados

O CSManager pode carregar o banco de dados AMCSD incluído e o banco de dados COD opcional, obtido conforme necessário.

## Bancos de dados padrão

- `Ler banco de dados AMCSD (### cristais)` carrega o conjunto de dados AMCSD incluído.
- `Ler banco de dados COD (### cristais)` carrega os dados do COD. O primeiro uso pode exigir o download de um grande arquivo do COD.

## Bancos de dados personalizados

- `Ler banco de dados` abre um arquivo de banco de dados do CSManager existente.
- `Salvar banco de dados` salva o banco de dados atual.
- `Limpar banco de dados` limpa os registros carregados sem excluir os arquivos do banco de dados.
