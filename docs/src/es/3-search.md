# Búsqueda

El panel de búsqueda permite filtrar la base de datos de cristales cargada por nombre, composición, simetría, constantes de red, valor d, densidad e información bibliográfica.

## Contenido previsto

- Búsqueda de texto en nombres y referencias.
- Especificación de elementos obligatorios/excluidos mediante el diálogo de tabla periódica.
- Filtrado por sistema cristalino y grupo espacial.
- Especificación de intervalos de constantes de red.
- Búsqueda por valor d, considerando o no el factor de dispersión.
- Especificación de intervalos de densidad.
