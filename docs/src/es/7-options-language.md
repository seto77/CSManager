# Opciones e idioma

## Opciones

El menú `Opciones` contiene los siguientes ajustes.

- `Información sobre herramientas` — Si se activa, muestra información sobre herramientas (un globo explicativo) en cada control.
- `Leer base de datos AMCSD en el próximo inicio` — Si se activa, carga la base de datos AMCSD al iniciar el programa. Está activado de forma predeterminada.
- `Búsqueda incremental` — Si se activa, refleja los resultados de la búsqueda de inmediato en cuanto se modifica un criterio de búsqueda.
- `Tamaño de fuente en la tabla de datos` — Cambia el tamaño del texto de la tabla de la base de datos.

## Ayuda

El menú `Ayuda` contiene los siguientes elementos.

- `Actualizaciones del programa` — Comprueba si se ha publicado una versión nueva y, en tal caso, la instala.
- `Historial de versiones` — Muestra el historial de versiones anteriores.
- `Licencia` — Muestra la licencia.
- `Página de Github` — Abre el repositorio de CSManager en GitHub.
- `Informar errores, solicitudes o comentarios` — Abre la página de Issues de GitHub.
- `Ayuda (PDF)` — Abre este manual en línea en el navegador predeterminado.

## Idioma

Desde el menú `Idioma` puede cambiar el idioma de la interfaz. El cambio requiere reiniciar el programa (el reinicio se realiza automáticamente). Además de inglés y japonés, CSManager admite alemán, francés, español, portugués, italiano, ruso, chino (simplificado y tradicional) y coreano.
