# Solución de problemas

Esta página recopila los problemas más frecuentes y sus soluciones.

## Contenido previsto

- Fallos en la descarga o la carga de COD.
- Caso en que no se encuentra .NET Desktop Runtime.
- Caso en que no se puede abrir el archivo de la base de datos.
- Caso en que la búsqueda es lenta con bases de datos grandes.
- Caso en que el intercambio de datos con PDIndexer / ReciPro no funciona correctamente.
