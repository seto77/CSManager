# Tabla de la base de datos

La tabla de la base de datos muestra las entradas de cristales cargadas. Al seleccionar una fila, se actualiza el editor de información de cristales situado en la parte inferior.

## Contenido previsto

- Ordenación de columnas.
- Conmutación de la visualización entre AMCSD y COD.
- Interpretación de las columnas de fórmula química, densidad, constantes de red, simetría y referencia.
- Envío del cristal seleccionado a las aplicaciones relacionadas.
