# Importación y exportación

CSManager puede importar cristales desde archivos CIF, AMC y desde las listas de cristales de PDIndexer / ReciPro, y puede exportar como CIF el cristal seleccionado.

## Importación

- `Importar CIF, AMC …` del menú `Archivo` — Lee un archivo CIF o AMC y lo añade al final de la base de datos actual.
- `Importar datos de cristales PDI (*.xml)` del menú `Archivo` — Lee una lista de cristales (`*.xml`) de PDIndexer o ReciPro y la añade al final de la base de datos actual.
- También puede cargar cristales arrastrando y soltando un archivo CIF / AMC en el área de [información de cristales](4-crystal-information/index.md). Asimismo, puede importarlos haciendo clic con el botón derecho en la información de cristales y eligiendo `Importar desde CIF o AMC...`.

CIF es el formato estándar de la Unión Internacional de Cristalografía (IUCr) y AMC es el formato adoptado por la Mineralogical Society of America.

## Exportación

- Al hacer clic con el botón derecho en la [información de cristales](4-crystal-information/index.md) y elegir `Exportar a CIF`, puede exportar el cristal actual en formato CIF.
- Para guardar la base de datos completa, use `Guardar base de datos` del menú `Archivo` (consulte [Carga de bases de datos](1-database-loading.md)).
