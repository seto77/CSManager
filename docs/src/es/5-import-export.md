# Importación y exportación

CSManager puede importar datos de cristales desde CIF, AMC y archivos XML de PDIndexer/ReciPro. También admite la exportación del cristal seleccionado como CIF y el guardado de bases de datos editadas.

## Contenido previsto

- `Importar CIF, AMC …`
- `Importar datos de cristales PDI (*.xml)`
- Arrastrar y soltar en el área de información de cristales.
- Guardado de bases de datos.
- Exportación de los cristales seleccionados.
