# Carga de bases de datos

La lectura y el guardado de las bases de datos se realizan desde el menú `Archivo`.

## Bases de datos estándar

- `Leer base de datos AMCSD (### cristales)` — Lee la base de datos AMCSD proporcionada por la Mineralogical Society of America (MSA). Contiene unas 20 000 estructuras cristalinas y, al venir incluida en el archivo de instalación, no es necesario descargarla por separado. El menú muestra el número de registros actuales. Si en `Opciones` ha activado «Leer base de datos AMCSD en el próximo inicio», se cargará automáticamente al arrancar.
- `Leer base de datos COD (### cristales)` — Lee la base de datos cristalográfica proporcionada por la Crystallography Open Database (COD). Contiene más de 500 000 estructuras cristalinas y, debido a su gran tamaño de archivo (más de 800 MB), no viene incluida. En la primera lectura se descarga automáticamente de Internet y, a partir de entonces, se lee el archivo ya descargado.

## Bases de datos personalizadas

- `Leer base de datos` — Abre cualquier archivo de base de datos de CSManager (`*.cdb3`).
- `Guardar base de datos` — Guarda la base de datos actual en formato `*.cdb3`.
- `Borrar base de datos` — Borra los datos ya cargados. El archivo de base de datos en sí no se elimina.

Si utiliza AMCSD o COD tal cual, apenas tendrá ocasión de usar `Leer base de datos` / `Guardar base de datos`.

## Uso y citación de la base de datos

Si publica artículos académicos u otros trabajos utilizando los datos de estructuras cristalinas incluidos en AMCSD, debe citar siempre el siguiente artículo, tanto en línea como sin conexión.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
