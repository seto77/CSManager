# Carga de bases de datos

CSManager puede cargar la base de datos AMCSD incluida y la base de datos COD, que se obtiene cuando es necesario.

## Bases de datos estándar

- `Leer base de datos AMCSD (### cristales)` carga el conjunto de datos AMCSD incluido.
- `Leer base de datos COD (### cristales)` carga los datos de COD. El primer uso puede requerir la descarga de un archivo COD de gran tamaño.

## Bases de datos personalizadas

- `Leer base de datos` abre un archivo de base de datos de CSManager existente.
- `Guardar base de datos` guarda la base de datos actual.
- `Borrar base de datos` borra las entradas cargadas. El propio archivo de la base de datos no se elimina.
