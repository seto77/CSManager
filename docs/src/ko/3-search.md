# 검색

검색 패널에서는 읽어 들인 결정 데이터베이스를 다양한 조건으로 좁힐 수 있습니다. 사용하려는 조건의 체크박스를 활성화하고 값을 입력한 다음, `검색` 버튼을 누르거나 Enter 키를 누르면 검색이 실행됩니다. 여러 조건을 동시에 지정할 수 있습니다.

## 검색 조건

- `Name` — 결정의 명칭으로 검색합니다.
- `원소` — `주기율표` 버튼을 누르면 별도의 윈도우가 열려 검색 대상 원소를 선택할 수 있습니다. 각 원소의 버튼은 누를 때마다 상태(포함해도 됨 / 반드시 포함 / 반드시 제외)가 전환됩니다. 윈도우 상단의 "may or not include" / "must include" / "must exclude" 버튼으로 모든 원소의 상태를 일괄 전환할 수 있습니다.
- `참고문헌` — 논문명, 학술지명, 저자명 등의 문헌 정보로 검색합니다.
- `결정계` — 결정계로 좁힙니다.
- `격자 상수` — 격자 상수와 허용 범위를 지정합니다. 허용 범위에 0을 입력한 항목은 무시됩니다.
- `면간 거리` — 결정면의 d 값(면간 거리)과 허용 범위를 지정하여, 그 범위에 포함되는 d 값을 갖는 결정을 검색합니다. `산란 인자 무시`를 활성화한 경우에는 회절 강도(결정 구조 인자)를 고려하지 않고, 소멸칙에 저촉되는 결정면도 포함하여 검색합니다. 비활성화한 경우에는 결정 구조 인자를 고려하여, 강도 상위 20개까지의 결정면만을 대상으로 검색합니다.
- `밀도` — 밀도와 허용 범위를 지정합니다.
