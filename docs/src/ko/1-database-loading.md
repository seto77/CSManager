# 데이터베이스 읽기

데이터베이스의 읽기/저장은 `파일` 메뉴에서 합니다.

## 표준 데이터베이스

- `AMCSD 데이터베이스 읽기 (### 개 결정 포함)` — 미국 광물학회 (MSA)가 제공하는 AMCSD 데이터베이스를 읽습니다. 약 2만 건의 결정 구조를 수록하고 있으며, 설치 파일에 동봉되어 있으므로 별도 다운로드는 필요하지 않습니다. 메뉴에는 현재 수록 건수가 표시됩니다. `옵션`에서 "다음 부팅 시 AMCSD 데이터베이스 읽기"를 활성화한 경우에는 실행 시 자동으로 읽힙니다.
- `COD 데이터베이스 읽기 (### 개 결정 포함)` — Crystallography Open Database (COD)가 제공하는 결정 데이터베이스를 읽습니다. 50만 건이 넘는 결정 구조를 수록하고 있으며, 파일 크기가 크기 때문에 (800 MB 초과) 동봉되어 있지 않습니다. 최초 읽기 시 인터넷에서 자동으로 다운로드하고, 다음번부터는 다운로드된 파일을 읽습니다.

## 사용자 지정 데이터베이스

- `데이터베이스 읽기` — 임의의 CSManager 데이터베이스 파일 (`*.cdb3`)을 엽니다.
- `데이터베이스 저장` — 현재 데이터베이스를 `*.cdb3` 형식으로 저장합니다.
- `데이터베이스 지우기` — 읽어 들인 데이터를 지웁니다. 데이터베이스 파일 자체는 삭제되지 않습니다.

AMCSD 또는 COD를 그대로 사용하는 경우, `데이터베이스 읽기` / `데이터베이스 저장`을 사용할 일은 거의 없습니다.

## 데이터베이스의 이용과 인용

AMCSD에 포함된 결정 구조 데이터를 사용하여 학술 논문 등을 출판하는 경우에는, 온라인/오프라인을 불문하고 반드시 다음 논문을 인용해 주십시오.

> Downs, R.T. and Hall-Wallace, M. (2003) The American Mineralogist Crystal Structure Database. *American Mineralogist* 88, 247-250.
