# 데이터베이스 읽기

CSManager 는 번들된 AMCSD 데이터베이스와, 필요에 따라 가져오는 COD 데이터베이스를 읽을 수 있습니다.

## 표준 데이터베이스

- `AMCSD 데이터베이스 읽기 (### 개 결정 포함)` 는 동봉된 AMCSD 데이터셋을 읽어 들입니다.
- `COD 데이터베이스 읽기 (### 개 결정 포함)` 는 COD 데이터를 읽어 들입니다. 처음 사용할 때는 대용량 COD 아카이브의 다운로드가 필요할 수 있습니다.

## 사용자 지정 데이터베이스

- `데이터베이스 읽기` 는 기존 CSManager 데이터베이스 파일을 엽니다.
- `데이터베이스 저장` 은 현재 데이터베이스를 저장합니다.
- `데이터베이스 지우기` 는 읽어 들인 항목을 지웁니다. 데이터베이스 파일 자체는 삭제되지 않습니다.
