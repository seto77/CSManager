# 메인 윈도우

CSManager를 실행하면 메인 윈도우가 표시됩니다. 윈도우는 크게 다음 영역으로 구성되어 있습니다.

- **메뉴(최상단)** — `파일`, `옵션`, `도움말`, `언어`의 각 메뉴. 데이터베이스의 읽기/쓰기와 앱 설정을 합니다.
- **결정 리스트(중앙부)** — 읽어 들인 결정의 목록. 행을 선택하면 하단의 결정 정보에 반영됩니다.
- **결정 정보(하단)** — 선택 중인 결정의 격자 상수, 대칭성, 원자 위치 등을 표시하고 편집합니다.
- **검색 패널(하단 오른쪽)** — 읽어 들인 데이터베이스를 다양한 조건으로 좁힙니다.
- **상태 영역(최하단)** — 읽기 진행 상황이나 처리 결과 메시지를 표시합니다.

각 영역의 자세한 사용법은 다음 페이지에서 설명합니다.
