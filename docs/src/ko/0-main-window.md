# 메인 윈도우

메인 윈도우는 결정 리스트, 선택 중인 결정 정보, 검색 패널을 중심으로 구성되어 있습니다.

## 영역

- 상단의 파일 메뉴와 애플리케이션 메뉴.
- 중앙의 결정 리스트.
- 선택한 결정을 확인·편집하는 결정 정보 편집기.
- 읽어 들인 데이터베이스를 좁혀 주는 검색 패널.
- 진행 상황이나 메시지를 표시하는 상태 영역.

## 편집 메모

- UI 라벨은 `CSManager/FormMain.ko.resx` 를 정본으로 삼아 확인한다.
- 스크린샷은 `docs/src/assets/cap-ko-auto/` 또는 `docs/src/assets/cap-ko-manual/` 로 옮긴다.
