# 옵션과 언어

## 옵션

`옵션` 메뉴에는 다음 설정이 있습니다.

- `도구 설명` — 활성화하면 각 컨트롤에 도구 설명(설명 말풍선)을 표시합니다.
- `다음 부팅 시 AMCSD 데이터베이스 읽기` — 활성화하면 실행과 동시에 AMCSD 데이터베이스를 읽어 들입니다. 기본적으로 활성화되어 있습니다.
- `점진 검색` — 활성화하면 검색 조건을 변경한 시점에 즉시 검색 결과에 반영합니다.
- `데이터 표 글꼴 크기` — 결정 리스트의 문자 크기를 변경합니다.

## 도움말

`도움말` 메뉴에는 다음 항목이 있습니다.

- `프로그램 업데이트` — 새로운 버전이 릴리스되어 있는지 확인하고, 있으면 업데이트합니다.
- `버전 기록` — 지금까지의 버전 기록을 표시합니다.
- `라이선스` — 라이선스를 표시합니다.
- `Github 페이지` — GitHub 상의 CSManager 리포지토리를 엽니다.
- `버그, 요청 또는 의견 보내기` — GitHub의 Issue 페이지를 엽니다.
- `도움말 (PDF)` — 이 온라인 매뉴얼을 기본 브라우저에서 엽니다.

## 언어

`언어` 메뉴에서 표시 언어를 전환할 수 있습니다. 전환에는 재시작이 필요합니다(재시작은 자동으로 이루어집니다). CSManager는 영어, 일본어 외에 독일어, 프랑스어, 스페인어, 포르투갈어, 이탈리아어, 러시아어, 중국어(간체자/번체자), 한국어를 지원합니다.
