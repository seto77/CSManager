# CSManager 매뉴얼

CSManager 는 결정 구조 데이터베이스를 열람·검색·편집하기 위한 Windows 데스크톱 애플리케이션입니다. 번들된 AMCSD 데이터베이스 읽기, COD 의 온디맨드 가져오기, 결정 파일 가져오기, 결정 정보 편집, 관련 소프트웨어와의 데이터 연동을 지원합니다.

이 GitHub Pages 판 매뉴얼을 앞으로의 정본으로 관리합니다. 초기 페이지는 기존 PDF/매뉴얼 구성과 현행 UI 를 바탕으로 골격을 마련했으며, 상세 본문은 Markdown 쪽에 직접 추가해 나갑니다.
