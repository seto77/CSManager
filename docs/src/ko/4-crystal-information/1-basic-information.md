# 기본 정보

`기본 정보` 탭에서는 결정의 격자 상수와 대칭성을 표시하고 설정합니다.

## 격자 상수와 대칭성

- **Cell constants** — 격자 상수를 표시하고 설정합니다. 길이의 단위는 Å (10⁻¹⁰ m)입니다. 대칭성을 지정하여 격자 상수에 제약이 있는 경우(예: 입방정의 a = b = c, α = β = γ = 90°)에는 자동으로 재설정됩니다.
- **Symmetry** — 대칭성을 계층별로 설정합니다. `Crystal system`(결정계), `Point group`(점군), `Space group`(공간군)을 지정합니다. 공간군은 축 설정에 주의하십시오. `Search`에 공간군 문자열을 입력하면 후보가 오른쪽 목록에 표시됩니다(대문자/소문자를 구별합니다).

## 산출되는 값

원자 정보가 입력되어 있는 경우, 다음 값이 계산되어 표시됩니다.

- **Cell Volume** — 단위 격자의 체적.
- **Cell Mass** — 단위 격자의 질량.
- **Molar Volume** / **Molar Mass** — 1 몰당 체적/질량.
- **Z** — 단위 격자에 포함되는 조성식의 수.
- **Density** — 밀도.
- **Color of Profile** — 회절 피크를 그리는 색. 클릭하면 색 설정 윈도우가 열립니다.

더 자세한 대칭성 정보(라우에군/점군의 각종 표기, 출현칙, Wyckoff position, 결정면/정대축의 기하 계산)는 우클릭 메뉴의 `대칭 정보`에서 확인할 수 있습니다.
