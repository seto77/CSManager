# 결합과 배위 다면체

`결합 & 다면체` 탭에서는 결정 구조를 3차원 표시할 때의 결합과 배위 다면체의 그리기를 설정합니다. 여기서 정의한 결합/다면체는 ReciPro 등의 구조 뷰어에서의 표시에 사용됩니다.

## 결합의 정의

- `결합 원자` — 결합시킬 2종류의 원자(중심 원자와 꼭짓점 원자)를 선택합니다.
- `추가` / `교체` / `삭제` — 설정한 결합을 목록에 추가, 선택 중인 결합을 교체, 또는 삭제합니다.

## 표시 옵션

- `결합 표시` — 원자 간의 결합을 표시합니다.
- `다면체 표시` — 배위 다면체를 표시합니다.
- `모서리 표시` — 다면체의 능선(모서리)을 표시합니다.
- `꼭짓점 원자 표시` / `중심 원자 표시` — 다면체의 꼭짓점 원자/중심 원자를 표시합니다.
- `내부 결합 표시` — 다면체 내부의 결합을 표시합니다.

다면체의 투명도 등의 세밀한 그리기 설정도 이 탭에서 조정할 수 있습니다.
