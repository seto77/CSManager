# 가져오기와 내보내기

CSManager는 CIF, AMC, PDIndexer / ReciPro의 결정 리스트에서 결정을 가져올 수 있으며, 선택 중인 결정을 CIF로 내보낼 수 있습니다.

## 가져오기

- `파일` 메뉴의 `CIF, AMC … 가져오기` — CIF 파일이나 AMC 파일을 읽어 들여 현재 데이터베이스의 끝에 추가합니다.
- `파일` 메뉴의 `PDI 결정 데이터 가져오기 (*.xml)` — PDIndexer나 ReciPro의 결정 리스트 (`*.xml`)를 읽어 들여 현재 데이터베이스의 끝에 추가합니다.
- [결정 정보](4-crystal-information/index.md) 영역에 CIF / AMC 파일을 드래그 앤 드롭해도 결정을 읽어 들일 수 있습니다. 또한 결정 정보를 우클릭하여 `CIF 또는 AMC에서 가져오기...`를 선택해도 가져올 수 있습니다.

CIF는 국제결정학연합 (IUCr)의 표준 포맷이며, AMC는 미국 광물학회가 채용하는 포맷입니다.

## 내보내기

- [결정 정보](4-crystal-information/index.md)를 우클릭하여 `CIF로 내보내기`를 선택하면 현재 결정을 CIF 형식으로 내보낼 수 있습니다.
- 데이터베이스 전체를 저장하려면 `파일` 메뉴의 `데이터베이스 저장`을 사용합니다([데이터베이스 읽기](1-database-loading.md) 참조).
