# 결정 리스트

데이터베이스를 읽어 들이면 메인 윈도우의 중앙부에 결정 목록(결정 리스트)이 표시됩니다. 각 행이 1건의 결정에 대응하며, 명칭, 화학 조성, 대칭성, 격자 상수, 밀도, 문헌 등의 정보가 열에 표시됩니다.

## 선택

결정을 선택하면 그 결정의 상세 정보가 하단의 [결정 정보](4-crystal-information/index.md)에 표시됩니다. 아울러 PDIndexer나 ReciPro가 실행되어 있는 경우에는, 선택과 동시에 그 결정의 정보가 클립보드를 통해 이 소프트웨어들로 전송됩니다([다른 소프트웨어와의 연동](6-interoperation.md) 참조).

## 정렬

각 열의 머리글(최상단)을 클릭하면 그 항목으로 오름차순/내림차순으로 정렬됩니다.
